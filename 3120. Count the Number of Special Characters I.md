# Problem Statement

You are given a string `word`.

A letter is called **special** if it appears in both:
- lowercase
- uppercase

Return the number of special letters in `word`.

---

## 🧪 Examples

### Example 1
```text
Input:
word = "aaAbcBC"

Output:
3
```

Explanation:
```text
Special characters:
'a', 'b', 'c'
```

---

### Example 2
```text
Input:
word = "abc"

Output:
0
```

Explanation:
```text
No uppercase letters exist.
```

---

### Example 3
```text
Input:
word = "abBCab"

Output:
1
```

Explanation:
```text
Only 'b' appears in both lowercase and uppercase.
```

---

## 📌 Constraints

```text
1 <= word.length <= 50
word consists only of English letters
```

---

## 💻 Solution (Python)

```python
class Solution:
    def numberOfSpecialChars(self, word):

        lower = set()
        upper = set()

        for ch in word:

            if ch.islower():
                lower.add(ch)
            else:
                upper.add(ch)

        count = 0

        for ch in lower:

            if ch.upper() in upper:
                count += 1

        return count
```
