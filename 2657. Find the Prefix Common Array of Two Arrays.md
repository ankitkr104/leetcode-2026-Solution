# Problem Statement

You are given two 0-indexed integer permutations `A` and `B` of length `n`.

A prefix common array `C` is defined such that:

```text
C[i] = count of numbers present in both A[0..i] and B[0..i]
```

Return the prefix common array `C`.

---

## 🧪 Examples

### Example 1
```text
Input:
A = [1,3,2,4]
B = [3,1,2,4]

Output:
[0,2,3,4]
```

Explanation:
```text
i = 0 → no common numbers
i = 1 → {1,3} common
i = 2 → {1,2,3} common
i = 3 → {1,2,3,4} common
```

---

### Example 2
```text
Input:
A = [2,3,1]
B = [3,1,2]

Output:
[0,1,3]
```

---

## 📌 Constraints

```text
1 <= A.length == B.length <= 50
1 <= A[i], B[i] <= n
A and B are permutations
```

---

## 🧠 Approach

### 💡 Key Insight

Track numbers already seen in:
- `A`
- `B`

Whenever:
- a number from A appears in B before
- or a number from B appears in A before

increase common count.

---

## 🔁 Algorithm

For each index `i`:
1. Mark `A[i]` as seen
2. Mark `B[i]` as seen
3. Check:
```text
if A[i] already appeared in B
if B[i] already appeared in A
```
4. Update answer

---
## 💻 Solution (Python)

```python
class Solution:
    def findThePrefixCommonArray(self, A, B):

        n = len(A)

        freq = [0] * (n + 1)
        ans = []

        common = 0

        for i in range(n):

            freq[A[i]] += 1
            if freq[A[i]] == 2:
                common += 1

            freq[B[i]] += 1
            if freq[B[i]] == 2:
                common += 1

            ans.append(common)

        return ans
```
