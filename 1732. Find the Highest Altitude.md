# Problem Statement

A biker starts a road trip at altitude:

```text
0
```

You are given an integer array:

```text
gain
```

where:

```text
gain[i]
```

represents the change in altitude between point `i` and point `i + 1`.

Return the **highest altitude** reached during the trip.

---

## 🧪 Example 1

```text
Input:
gain = [-5,1,5,0,-7]

Output:
1
```

### Explanation

Starting altitude:

```text
0
```

Altitudes become:

```text
0
-5
-4
1
1
-6
```

Highest altitude:

```text
1
```

---

## 🧪 Example 2

```text
Input:
gain = [-4,-3,-2,-1,4,3,2]

Output:
0
```

### Explanation

Altitudes:

```text
0
-4
-7
-9
-10
-6
-3
-1
```

Highest altitude:

```text
0
```

---

## 📌 Constraints

```text
1 <= gain.length <= 100

-100 <= gain[i] <= 100
```
