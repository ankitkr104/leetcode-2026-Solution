# Problem Statement

You are given:
- a binary string `s`
- integers `minJump` and `maxJump`

Initially, you stand at index `0`.

You can jump from index `i` to index `j` if:

```text
i + minJump <= j <= i + maxJump
```

AND:
```text
s[j] == '0'
```

Return:
```text
true
```
if you can reach the last index, otherwise:
```text
false
```

---

## 🧪 Examples

### Example 1
```text
Input:
s = "011010"
minJump = 2
maxJump = 3

Output:
true
```

Explanation:
```text
0 → 3 → 5
```

---

### Example 2
```text
Input:
s = "01101110"
minJump = 2
maxJump = 3

Output:
false
```

---

## 📌 Constraints

```text
2 <= s.length <= 10^5
s[i] is either '0' or '1'
s[0] == '0'
1 <= minJump <= maxJump < s.length
```

---

## 🧠 Approach

### 💡 Key Insight

Naive BFS checking every possible jump:
```text
O(n * maxJump)
```

would cause:
```text
TLE
```

We optimize using:
```text
Sliding Window + BFS
```
