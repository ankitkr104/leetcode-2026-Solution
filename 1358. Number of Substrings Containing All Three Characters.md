## Problem Statement

You are given a string `s` consisting only of the characters:

```text
'a', 'b', and 'c'
```

Return the number of substrings that contain **at least one occurrence** of all three characters.

---

## Example 1

```text
Input:
s = "abcabc"

Output:
10
```

### Explanation

Valid substrings are:

```text
abc
abca
abcab
abcabc
bca
bcab
bcabc
cab
cabc
abc
```

Total:

```text
10
```

---

## Example 2

```text
Input:
s = "aaacb"

Output:
3
```

Valid substrings:

```text
aaacb
aacb
acb
```

---

## Example 3

```text
Input:
s = "abc"

Output:
1
```

Only one valid substring:

```text
abc
```

---

## Constraints

```text
3 <= s.length <= 5 × 10^4

s contains only
'a', 'b', 'c'
```

---

# Python Solution

```python
class Solution:
    def numberOfSubstrings(self, s):

        count = {'a': 0, 'b': 0, 'c': 0}

        left = 0

        ans = 0

        n = len(s)

        for right in range(n):

            count[s[right]] += 1

            while (
                count['a'] > 0 and
                count['b'] > 0 and
                count['c'] > 0
            ):

                ans += n - right

                count[s[left]] -= 1

                left += 1

        return ans
```
