## Problem Statement

You are given a string `s` consisting only of the characters:

```text
'a', 'b', and 'c'
```

Return the number of substrings that contain **at least one occurrence** of all three characters.

---

## Example 1

```text
Input:
s = "abcabc"

Output:
10
```

### Explanation

Valid substrings are:

```text
abc
abca
abcab
abcabc
bca
bcab
bcabc
cab
cabc
abc
```

Total:

```text
10
```

---

## Example 2

```text
Input:
s = "aaacb"

Output:
3
```

Valid substrings:

```text
aaacb
aacb
acb
```

---

## Example 3

```text
Input:
s = "abc"

Output:
1
```

Only one valid substring:

```text
abc
```

---

## Constraints

```text
3 <= s.length <= 5 × 10^4

s contains only
'a', 'b', 'c'
```
