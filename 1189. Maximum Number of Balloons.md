# Problem Statement

Given a string:

```text
text
```

You want to form as many instances of the word:

```text
"balloon"
```

as possible.

Each character in `text` can be used **at most once**.

Return the maximum number of times the word:

```text
"balloon"
```

can be formed.

---

## 🧪 Example 1

```text
Input:
text = "nlaebolko"

Output:
1
```

### Explanation

Characters available:

```text
b = 1
a = 1
l = 2
o = 2
n = 1
```

Can form:

```text
"balloon"
```

exactly once.

---

## 🧪 Example 2

```text
Input:
text = "loonbalxballpoon"

Output:
2
```

### Explanation

There are enough characters to form:

```text
"balloon"
```

twice.

---

## 🧪 Example 3

```text
Input:
text = "leetcode"

Output:
0
```

### Explanation

Missing required characters for:

```text
"balloon"
```

---

## 📌 Constraints

```text
1 <= text.length <= 10^4

text consists of lowercase English letters.
```

---

## 💻 Solution (Python)

```python
from collections import Counter

class Solution:
    def maxNumberOfBalloons(self, text):

        cnt = Counter(text)

        return min(
            cnt['b'],
            cnt['a'],
            cnt['l'] // 2,
            cnt['o'] // 2,
            cnt['n']
        )
```
