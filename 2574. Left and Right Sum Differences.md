# Problem Statement

You are given a 0-indexed integer array `nums` of size `n`.

Define:

### Left Sum

```text
leftSum[i]
```

= sum of all elements to the left of index `i`.

If no such element exists:

```text
leftSum[i] = 0
```

---

### Right Sum

```text
rightSum[i]
```

= sum of all elements to the right of index `i`.

If no such element exists:

```text
rightSum[i] = 0
```

Return an array:

```text
answer[i] = |leftSum[i] - rightSum[i]|
```

---

## 🧪 Examples

### Example 1

```text
Input:
nums = [10,4,8,3]

Output:
[15,1,11,22]
```

### Explanation

```text
leftSum  = [0,10,14,22]
rightSum = [15,11,3,0]

answer =
[
|0-15|,
|10-11|,
|14-3|,
|22-0|
]

= [15,1,11,22]
```

---

### Example 2

```text
Input:
nums = [1]

Output:
[0]
```

### Explanation

```text
leftSum  = [0]
rightSum = [0]

answer = [0]
```

---

## 📌 Constraints

```text
1 <= nums.length <= 1000
1 <= nums[i] <= 10^5
```

---
## 💻 Solution (Python)

```python
class Solution:
    def leftRightDifference(self, nums):

        total = sum(nums)
        left_sum = 0
        ans = []

        for num in nums:

            right_sum = total - left_sum - num

            ans.append(
                abs(left_sum - right_sum)
            )

            left_sum += num

        return ans
```
