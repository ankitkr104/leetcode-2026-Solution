# Problem Statement

You are given two string arrays `queries` and `dictionary`.

- All words consist of **lowercase English letters**
- All words have the **same length**

---

### Task

In one edit, you can change **any one character** in a word to any other character.

Return all words from `queries` such that:
- They can be transformed into **any word in `dictionary`**
- Using **at most 2 edits**

Return the result in the **same order** as `queries`.

---

## Examples

### Example 1


Input:
queries = ["word","note","ants","wood"]
dictionary = ["wood","joke","moat"]

Output:
["word","note","wood"]


**Explanation:**
- `"word"` → 1 edit → `"wood"`
- `"note"` → 2 edits → `"joke"`
- `"ants"` → needs >2 edits ❌
- `"wood"` → 0 edits ✅

---

### Example 2


Input:
queries = ["yes"]
dictionary = ["not"]

Output:
[]


---

## Constraints

- `1 <= queries.length, dictionary.length <= 100`
- `1 <= word length <= 100`
- All words have equal length

---

## Solution

### 💻 Code

```python
class Solution:
    def twoEditWords(self, queries, dictionary):
        def is_valid(q, d):
            diff = 0
            for i in range(len(q)):
                if q[i] != d[i]:
                    diff += 1
                if diff > 2:
                    return False
            return True

        result = []

        for q in queries:
            for d in dictionary:
                if is_valid(q, d):
                    result.append(q)
                    break   # no need to check more dictionary words

        return result
