# Problem Statement

Given two integer arrays `nums1` and `nums2` sorted in non-decreasing order, return the minimum integer common to both arrays.

If there is no common integer, return:

```text
-1
```

---

## 🧪 Examples

### Example 1
```text
Input:
nums1 = [1,2,3]
nums2 = [2,4]

Output:
2
```

Explanation:
```text
2 is the smallest common element.
```

---

### Example 2
```text
Input:
nums1 = [1,2,3,6]
nums2 = [2,3,4,5]

Output:
2
```

---

## 📌 Constraints

```text
1 <= nums1.length, nums2.length <= 10^5
1 <= nums1[i], nums2[j] <= 10^9
Both arrays are sorted in non-decreasing order
```

---

## 🧠 Approach

### 💡 Key Insight

Since both arrays are already sorted,
use the:

```text
Two Pointer Technique
```

---

## 🔁 Two Pointer Logic

- If:
```text
nums1[i] == nums2[j]
```
→ found minimum common value

- If:
```text
nums1[i] < nums2[j]
```
→ move `i`

- Else:
```text
nums2[j] < nums1[i]
```
→ move `j`

This works because arrays are sorted.

---

## 💻 Solution (C++)

```cpp
class Solution {
public:
    int getCommon(vector<int>& nums1, vector<int>& nums2) {

        int n = nums1.size();
        int m = nums2.size();

        int i = 0;
        int j = 0;

        while(i < n && j < m)
        {
            if(nums1[i] == nums2[j])
                return nums1[i];

            else if(nums1[i] < nums2[j])
                i++;

            else
                j++;
        }

        return -1;
    }
};
```
