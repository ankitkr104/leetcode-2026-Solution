# Problem Statement

You are given:

- An integer array `nums` of length `n`.
- An integer array `queries`.

Construct an array `gcdPairs` containing the GCD of every pair

```text
(nums[i], nums[j])   where 0 ≤ i < j < n
```

Sort `gcdPairs` in **ascending order**.

For each query `queries[i]`, return the value located at index `queries[i]` in the sorted `gcdPairs` array.

The answer should be returned as an array.

---

## Example 1

```text
Input

nums = [2,3,4]
queries = [0,2,2]
```

Output

```text
[1,2,2]
```

### Explanation

All pairs

```text
(2,3) -> 1

(2,4) -> 2

(3,4) -> 1
```

Sorted

```text
[1,1,2]
```

Answer

```text
[1,2,2]
```

## Constraints

```text
2 <= n <= 100000

1 <= nums[i] <= 50000

1 <= queries.length <= 100000
```

# Python Solution

```python
from bisect import bisect_left

class Solution:
    def gcdValues(self, nums, queries):

        MAX = max(nums)

        freq = [0] * (MAX + 1)

        for x in nums:
            freq[x] += 1

        divCount = [0] * (MAX + 1)

        for g in range(1, MAX + 1):
            for multiple in range(g, MAX + 1, g):
                divCount[g] += freq[multiple]

        exact = [0] * (MAX + 1)

        for g in range(MAX, 0, -1):

            c = divCount[g]

            exact[g] = c * (c - 1) // 2

            multiple = g * 2

            while multiple <= MAX:

                exact[g] -= exact[multiple]

                multiple += g

        prefix = [0]

        for g in range(1, MAX + 1):
            prefix.append(prefix[-1] + exact[g])

        ans = []

        for q in queries:
            ans.append(bisect_left(prefix, q + 1))

        return ans
```
