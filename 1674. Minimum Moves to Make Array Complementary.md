# Problem Statement

You are given:
- an integer array `nums` of even length `n`
- an integer `limit`

In one move, you can replace any number in `nums` with another integer between:

```text
1 to limit
```

An array is called **complementary** if:

```text
nums[i] + nums[n - 1 - i]
```

is the same for every index `i`.

Return the **minimum number of moves** required to make the array complementary.

---

## 🧪 Examples

### Example 1
```text
Input:
nums = [1,2,4,3]
limit = 4

Output:
1
```

Explanation:
```text
Change nums to [1,2,2,3]

1 + 3 = 4
2 + 2 = 4
```

---

### Example 2
```text
Input:
nums = [1,2,2,1]
limit = 2

Output:
2
```

---

### Example 3
```text
Input:
nums = [1,2,1,2]
limit = 2

Output:
0
```

---

## 📌 Constraints

```text
n == nums.length
2 <= n <= 10^5
1 <= nums[i] <= limit <= 10^5
n is even
```

---

## 🧠 Approach

### 💡 Key Insight

For every pair:

```text
(a, b)
```

Possible costs for making their sum equal to `target`:

| Condition | Moves |
|---|---|
| target = a + b | 0 |
| target achievable by changing one number | 1 |
| otherwise | 2 |

---

## 🔥 Optimization Trick

Instead of checking every target separately:

Use a **difference array** to track:
- where move counts increase/decrease

This reduces complexity from:

```text
O(n * limit)
```

to:

```text
O(n + limit)
```

---

## 🔁 Pair Analysis

For pair `(a, b)`:

```text
low = min(a, b) + 1
high = max(a, b) + limit
sum = a + b
```

### Move ranges

| Range | Moves |
|---|---|
| outside [low, high] | 2 |
| inside [low, high] | 1 |
| exactly sum | 0 |

---

## 💻 Solution (Python)

```python
class Solution:
    def minMoves(self, nums, limit):
        n = len(nums)

        diff = [0] * (2 * limit + 2)

        for i in range(n // 2):

            a = nums[i]
            b = nums[n - 1 - i]

            low = min(a, b) + 1
            high = max(a, b) + limit
            s = a + b

            # Default: 2 moves
            diff[2] += 2

            # One move range starts
            diff[low] -= 1

            # Zero move at exact sum
            diff[s] -= 1
            diff[s + 1] += 1

            # Back to 2 moves after high
            diff[high + 1] += 1

        ans = float('inf')
        curr = 0

        for target in range(2, 2 * limit + 1):
            curr += diff[target]
            ans = min(ans, curr)

        return ans
```
