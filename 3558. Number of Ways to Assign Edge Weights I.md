# Problem Statement

You are given an undirected tree rooted at node `1`.

Initially, every edge has weight:

```text
0
```

You must assign each edge a weight of:

```text
1 or 2
```

Choose any node `x` at the **maximum depth**.

Consider only the path:

```text
1 → x
```

Let the path contain `d` edges.

Return the number of ways to assign weights to these `d` edges such that:

```text
total path cost is odd
```

Return the answer modulo:

```text
10^9 + 7
```

---

## 🧪 Examples

### Example 1

```text
Input:
edges = [[1,2]]

Output:
1
```

Explanation:

```text
Path length = 1

Assignments:
1 → odd ✓
2 → even ✗

Answer = 1
```

---

### Example 2

```text
Input:
edges =
[[1,2],
 [1,3],
 [3,4],
 [3,5]]

Output:
2
```

Explanation:

```text
Maximum depth = 2

Path contains 2 edges.

Possible assignments:

(1,2) → 3 ✓
(2,1) → 3 ✓

Answer = 2
```

---

## 📌 Constraints

```text
2 <= n <= 10^5

edges.length == n - 1

1 <= ui, vi <= n

edges represents a valid tree
```

---

## 💻 Solution (Python)

```python
from collections import deque

class Solution:
    def assignEdgeWeights(self, edges):

        MOD = 10**9 + 7

        n = len(edges) + 1

        graph = [[] for _ in range(n + 1)]

        for u, v in edges:
            graph[u].append(v)
            graph[v].append(u)

        q = deque([(1, 0)])
        visited = {1}

        max_depth = 0

        while q:

            node, depth = q.popleft()

            max_depth = max(max_depth, depth)

            for nei in graph[node]:

                if nei not in visited:

                    visited.add(nei)
                    q.append((nei, depth + 1))

        return pow(2, max_depth - 1, MOD)
```
