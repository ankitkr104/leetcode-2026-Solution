# Problem Statement

You are given two arrays of positive integers `arr1` and `arr2`.

A prefix of an integer is formed by one or more digits starting from the leftmost digit.

Example:
```text
123 is a prefix of 12345
234 is NOT a prefix of 12345
```

Return the length of the **longest common prefix** among all pairs:

```text
(x, y)
where x ∈ arr1 and y ∈ arr2
```

If no common prefix exists, return:

```text
0
```

---

## 🧪 Examples

### Example 1
```text
Input:
arr1 = [1,10,100]
arr2 = [1000]

Output:
3
```

Explanation:
```text
1 and 1000 → common prefix = "1"
10 and 1000 → common prefix = "10"
100 and 1000 → common prefix = "100"
```

Longest length = 3

---

### Example 2
```text
Input:
arr1 = [1,2,3]
arr2 = [4,4,4]

Output:
0
```

---

## 📌 Constraints

```text
1 <= arr1.length, arr2.length <= 5 * 10^4
1 <= arr1[i], arr2[i] <= 10^8
```

---

## 🧠 Approach

### 💡 Key Insight

Store all prefixes from `arr1` in a set.

Then for every number in `arr2`:
- generate prefixes
- check if prefix exists in set
- maximize length
