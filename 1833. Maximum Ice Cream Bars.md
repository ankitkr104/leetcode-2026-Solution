# Problem Statement

A boy wants to buy as many ice cream bars as possible.

You are given:

```text
costs[i]
```

= cost of the `i-th` ice cream bar.

And:

```text
coins
```

= total coins available.

The boy can buy ice creams in **any order**.

Return the maximum number of ice cream bars he can buy.

> **Note:** The problem specifically asks for a **Counting Sort** solution.

---

## 🧪 Example 1

```text
Input:
costs = [1,3,2,4,1]
coins = 7

Output:
4
```

### Explanation

Buy:

```text
1 + 1 + 2 + 3 = 7
```

Total bars:

```text
4
```

---

## 🧪 Example 2

```text
Input:
costs = [10,6,8,7,7,8]
coins = 5

Output:
0
```

### Explanation

All ice creams cost more than available coins.

---

## 🧪 Example 3

```text
Input:
costs = [1,6,3,1,2,5]
coins = 20

Output:
6
```

### Explanation

Total cost:

```text
1 + 6 + 3 + 1 + 2 + 5 = 18
```

All ice creams can be purchased.

---

## 📌 Constraints

```text
1 <= costs.length <= 10^5

1 <= costs[i] <= 10^5

1 <= coins <= 10^8
```

---

## 💻 Solution (Python)

```python
class Solution:
    def maxIceCream(self, costs, coins):

        max_cost = max(costs)

        freq = [0] * (max_cost + 1)

        for cost in costs:
            freq[cost] += 1

        ans = 0

        for price in range(1, max_cost + 1):

            if freq[price] == 0:
                continue

            can_buy = min(
                freq[price],
                coins // price
            )

            ans += can_buy

            coins -= can_buy * price

            if coins < price:
                continue

        return ans
```
