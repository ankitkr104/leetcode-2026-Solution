# Problem Description

You are given an `m x n` grid where each cell contains one of the values `0`, `1`, or `2`. You are also given an integer `k`.

You start from the **top-left corner (0, 0)** and want to reach the **bottom-right corner (m - 1, n - 1)** by moving only **right** or **down**.

Each cell contributes:

| Value | Score | Cost |
|------|------|------|
| 0    | 0    | 0    |
| 1    | 1    | 1    |
| 2    | 2    | 1    |

### Goal
Return the **maximum score** achievable such that the **total cost ≤ k**.

If no valid path exists, return `-1`.

---

## Note

- If the total cost exceeds `k`, the path is **invalid**.
- Starting cell `(0,0)` always has value `0`.

---

## 🧪 Examples

### Example 1


Input:
grid = [[0,1],[2,0]]
k = 1

Output:
2


### Explanation

| Cell   | Value | Score | Total Score | Cost | Total Cost |
|--------|------|------|-------------|------|------------|
| (0,0)  | 0    | 0    | 0           | 0    | 0          |
| (1,0)  | 2    | 2    | 2           | 1    | 1          |
| (1,1)  | 0    | 0    | 2           | 0    | 1          |

✅ Maximum score = **2**

---

### Example 2


Input:
grid = [[0,1],[1,2]]
k = 1

Output:
-1


### Explanation

❌ No valid path exists within cost `k`.

---

## 🧠 Approach

This problem is a combination of:

- **Grid traversal**
- **Dynamic Programming**
- **Knapsack constraint (cost ≤ k)**

### 🔑 Idea

Define:


dp[i][j][c] = maximum score to reach cell (i,j) using cost c


We only move:
- Right → `(i, j-1)`
- Down → `(i-1, j)`

### Transition


dp[i][j][c] = max(
dp[i-1][j][c - cost] + value,
dp[i][j-1][c - cost] + value
)

---
## 💻 Solution

```python3
class Solution:
    def maxPathScore(self, grid, k):
        m, n = len(grid), len(grid[0])

        INF = float("-inf")
        dp = [[[INF] * (k + 1) for _ in range(n)] for _ in range(m)]
        dp[0][0][0] = 0

        for i in range(m):
            for j in range(n):
                for c in range(k + 1):
                    if dp[i][j][c] == INF:
                        continue

                    if i + 1 < m:
                        val = grid[i + 1][j]
                        cost = 0 if val == 0 else 1
                        if c + cost <= k:
                            dp[i + 1][j][c + cost] = max(
                                dp[i + 1][j][c + cost], dp[i][j][c] + val
                            )

                    if j + 1 < n:
                        val = grid[i][j + 1]
                        cost = 0 if val == 0 else 1
                        if c + cost <= k:
                            dp[i][j + 1][c + cost] = max(
                                dp[i][j + 1][c + cost], dp[i][j][c] + val
                            )

        ans = max(dp[m - 1][n - 1])
        return -1 if ans < 0 else ans
