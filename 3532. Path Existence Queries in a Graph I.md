# Problem Statement

You are given:

- An integer `n` representing the number of nodes labeled from `0` to `n-1`.
- A **sorted** array `nums` of length `n`.
- An integer `maxDiff`.

An undirected edge exists between nodes `i` and `j` if

```text
|nums[i] - nums[j]| ≤ maxDiff
```

You are also given several queries.

For each query

```text
[u, v]
```

determine whether there exists **any path** between nodes `u` and `v`.

Return an array of booleans.

---

## Example 1

```text
Input

n = 2

nums = [1,3]

maxDiff = 1

queries =
[[0,0],
 [0,1]]
```

Output

```text
[true,false]
```

Explanation

```text
0 → 0

Always reachable.
```

```text
|1-3| = 2 > 1
```

No edge exists between 0 and 1.

---

## Example 2

```text
Input

n = 4

nums = [2,5,6,8]

maxDiff = 2

queries =
[[0,1],
 [0,2],
 [1,3],
 [2,3]]
```

Output

```text
[false,false,true,true]
```

Explanation

Edges:

```text
1 -- 2

2 -- 3
```

Therefore

```text
1 → 3
```

is reachable through node 2.

---

## Constraints

```text
1 ≤ n ≤ 100000

nums is sorted

1 ≤ queries ≤ 100000
```

---

# Python Solution

```python
class Solution:
    def pathExistenceQueries(self, n, nums, maxDiff, queries):

        comp = [0] * n

        cid = 0

        for i in range(1, n):

            if nums[i] - nums[i - 1] > maxDiff:
                cid += 1

            comp[i] = cid

        ans = []

        for u, v in queries:

            ans.append(comp[u] == comp[v])

        return ans
```
