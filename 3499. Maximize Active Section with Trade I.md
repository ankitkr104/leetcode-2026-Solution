# Problem Statement

You are given a binary string `s` of length `n`, where:

- `'1'` represents an **active** section.
- `'0'` represents an **inactive** section.

You may perform **at most one trade** consisting of two steps:

1. Convert a contiguous block of `'1'`s that is **surrounded by `'0'`s** into all `'0'`s.
2. Then convert a contiguous block of `'0'`s that is **surrounded by `'1'`s** into all `'1'`s.

To determine whether a block is surrounded, consider the augmented string

```text
t = '1' + s + '1'
```

The added `'1'`s are only used for checking boundaries and are **not counted** in the final answer.

Return the maximum number of active sections after performing the optimal trade.

---

## Example 1

```text
Input

s = "01"
```

Output

```text
1
```

### Explanation

There is no valid block of `'1'`s surrounded by `'0'`s.

No trade can be made.

Answer

```text
1
```

---

## Example 2

```text
Input

s = "0100"
```

Output

```text
4
```

### Explanation

Augmented string

```text
101001
```

Trade

```text
101001

↓

100001

↓

111111
```

Removing the augmented ends gives

```text
1111
```

Answer

```text
4
```

## Constraints

```text
1 <= n <= 100000

s consists only of '0' and '1'
```
