# Problem Statement

Given an array of non-negative integers `arr`, you are initially positioned at index `start`.

From index `i`, you can jump to:

```text
i + arr[i]
```

or

```text
i - arr[i]
```

Return `true` if you can reach **any index with value 0**, otherwise return `false`.

⚠️ You cannot jump outside the array.

---

## 🧪 Examples

### Example 1
```text
Input:
arr = [4,2,3,0,3,1,2]
start = 5

Output:
true
```

Explanation:
```text
5 → 4 → 1 → 3
```

Index `3` contains `0`.

---

### Example 2
```text
Input:
arr = [4,2,3,0,3,1,2]
start = 0

Output:
true
```

---

### Example 3
```text
Input:
arr = [3,0,2,1,2]
start = 2

Output:
false
```

---

## 📌 Constraints

```text
1 <= arr.length <= 5 * 10^4
0 <= arr[i] < arr.length
0 <= start < arr.length
```

---

## 🧠 Approach

### 💡 Key Insight

This problem can be viewed as a graph:
- each index = node
- jumps = edges

We need to check whether a path exists to any node with value `0`.

Use:
```text
BFS or DFS
```

to explore reachable indices.

---

## 🔁 BFS Strategy

1. Start from `start`
2. Explore:
```text
i + arr[i]
i - arr[i]
```
3. Track visited indices
4. If any index contains `0` → return `true`
