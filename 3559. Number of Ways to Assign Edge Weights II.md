# Problem Statement

You are given a tree rooted at node `1`.

Each edge can be assigned either:

```text
1 or 2
```

For each query:

```text
[u, v]
```

consider only the edges on the path:

```text
u → v
```

Return the number of assignments such that:

```text
path cost is odd
```

where:

```text
path cost = sum of edge weights on the path
```

Return all answers modulo:

```text
10^9 + 7
```

---

## 🧪 Example 1

```text
Input:

edges = [[1,2]]

queries =
[[1,1],
 [1,2]]

Output:
[0,1]
```

### Explanation

```text
Path(1,1):
length = 0

cost = 0
No odd assignment

Answer = 0
```

```text
Path(1,2):
length = 1

Assignments:
1 → odd ✓
2 → even ✗

Answer = 1
```

```text

edges =
[[1,2],

[[1,4],
 [3,4],
 [2,5]]
```

## 🧠 Key Observation

Suppose path length is:

```text
d edges
```

Each edge can be:

```text
1 (odd)
2 (even)
```

Total assignments:

```text
2^d
```

For the sum to be odd:

```text
number of odd edges must be odd
```

Among all assignments:

```text
Exactly half
produce odd sum
```

Therefore:

```text
Answer = 2^(d-1)
```

for:

```text
d > 0
```

and

```text
Answer = 0
```

for:

```text
d = 0
```

---

## 💡 Remaining Problem

We only need to find:

```text
Path Length(u,v)
```

for up to:

```text
10^5 queries
```

---

## 🔥 Use LCA

Distance between two nodes:

```text
dist(u,v)
=
depth[u]
+
depth[v]
-
2 * depth[LCA(u,v)]
```

This gives the number of edges on the path.

---

## 🔁 Algorithm

### Preprocessing

1. Build tree.
2. DFS from root.
3. Store:
   - depth[]
   - binary lifting table up[][]

### For each query

Compute:

```text
d = distance(u,v)
```

If:

```text
d == 0
```

answer:

```text
0
```

Else:

```text
2^(d-1) mod M
```

---

## 💻 Solution (Python3)

```python
from typing import List
import math


class Solution:
    def assignEdgeWeights(
        self,
        edges: List[List[int]],
        queries: List[List[int]]
    ) -> List[int]:

        MOD = 10**9 + 7

        n = len(edges) + 1

        graph = [[] for _ in range(n + 1)]

        for u, v in edges:
            graph[u].append(v)
            graph[v].append(u)

        LOG = math.ceil(math.log2(n)) + 1

        up = [[0] * LOG for _ in range(n + 1)]
        depth = [0] * (n + 1)

        def dfs(node, parent):

            up[node][0] = parent

            for j in range(1, LOG):
                up[node][j] = up[
                    up[node][j - 1]
                ][j - 1]

            for nei in graph[node]:

                if nei == parent:
                    continue

                depth[nei] = depth[node] + 1

                dfs(nei, node)

        dfs(1, 0)

        def lca(a, b):

            if depth[a] < depth[b]:
                a, b = b, a

            diff = depth[a] - depth[b]

            for j in range(LOG):

                if diff & (1 << j):
                    a = up[a][j]

            if a == b:
                return a

            for j in range(LOG - 1, -1, -1):

                if up[a][j] != up[b][j]:
                    a = up[a][j]
                    b = up[b][j]

            return up[a][0]

        ans = []

        for u, v in queries:

            ancestor = lca(u, v)

            d = (
                depth[u]
                + depth[v]
                - 2 * depth[ancestor]
            )

            if d == 0:
                ans.append(0)
            else:
                ans.append(
                    pow(2, d - 1, MOD)
                )

        return ans
```
