# Problem Statement

You are given an integer array `nums`.

A **XOR triplet** is defined as

```text
nums[i] XOR nums[j] XOR nums[k]
```

where

```text
i <= j <= k
```

Return the number of **distinct XOR values** that can be obtained from all possible triplets.

---

## Example 1

```text
Input

nums = [1,3]
```

Output

```text
2
```

### Explanation

Possible XOR values

```text
1 XOR 1 XOR 1 = 1

1 XOR 1 XOR 3 = 3

1 XOR 3 XOR 3 = 1

3 XOR 3 XOR 3 = 3
```

Unique values

```text
{1,3}
```

Answer

```text
2
```

---

## Example 2

```text
Input

nums = [6,7,8,9]
```

Output

```text
4
```

### Explanation

The unique XOR triplet values are

```text
{6,7,8,9}
```

Answer

```text
4
```

---

## Constraints

```text
1 <= nums.length <= 1500

1 <= nums[i] <= 1500
```

---

# Python3 Solution

```python
class Solution:
    def uniqueXorTriplets(self, nums):
        MAX = 2048

        dp1 = [False] * MAX
        dp2 = [False] * MAX
        dp3 = [False] * MAX

        for x in nums:
            dp1[x] = True

        for x in nums:
            nxt = dp2[:]
            for v in range(MAX):
                if dp1[v]:
                    nxt[v ^ x] = True
            dp2 = nxt

        for x in nums:
            nxt = dp3[:]
            for v in range(MAX):
                if dp2[v]:
                    nxt[v ^ x] = True
            dp3 = nxt

        return sum(dp3)
```
