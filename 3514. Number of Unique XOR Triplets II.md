# Problem Statement

You are given an integer array `nums`.

A **XOR triplet** is defined as

```text
nums[i] XOR nums[j] XOR nums[k]
```

where

```text
i <= j <= k
```

Return the number of **distinct XOR values** that can be obtained from all possible triplets.

---

## Example 1

```text
Input

nums = [1,3]
```

Output

```text
2
```

### Explanation

Possible XOR values

```text
1 XOR 1 XOR 1 = 1

1 XOR 1 XOR 3 = 3

1 XOR 3 XOR 3 = 1

3 XOR 3 XOR 3 = 3
```

Unique values

```text
{1,3}
```

Answer

```text
2
```

---

## Example 2

```text
Input

nums = [6,7,8,9]
```

Output

```text
4
```

### Explanation

The unique XOR triplet values are

```text
{6,7,8,9}
```

Answer

```text
4
```

---

## Constraints

```text
1 <= nums.length <= 1500

1 <= nums[i] <= 1500
```
