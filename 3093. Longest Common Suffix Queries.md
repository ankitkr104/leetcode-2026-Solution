# Problem Statement

You are given:
- `wordsContainer`
- `wordsQuery`

For each query string:
- find the string in `wordsContainer`
- having the **longest common suffix**

If multiple strings qualify:
1. choose the shortest string
2. if still tied, choose the earliest index

Return an array containing the chosen indices.

---

## 🧪 Examples

### Example 1
```text
Input:
wordsContainer = ["abcd","bcd","xbcd"]
wordsQuery = ["cd","bcd","xyz"]

Output:
[1,1,1]
```

---

### Example 2
```text
Input:
wordsContainer = ["abcdefgh","poiuygh","ghghgh"]
wordsQuery = ["gh","acbfgh","acbfegh"]

Output:
[2,0,2]
```

---

## 📌 Constraints

```text
1 <= wordsContainer.length, wordsQuery.length <= 10^4
1 <= word length <= 5 * 10^3
Total characters <= 5 * 10^5
```

---

## 🧠 Approach

### 💡 Key Insight

Suffix problems can be converted into:
```text
Prefix problems
```

by reversing strings.

Example:
```text
"abcd" → "dcba"
```

Now:
```text
common suffix → common prefix
```

Use:
```text
Trie
```

for efficient matching.

---

## 💻 Solution (Python)

```python
class TrieNode:
    def __init__(self):
        self.children = {}
        self.idx = -1
        self.length = float('inf')


class Solution:
    def stringIndices(self, wordsContainer, wordsQuery):

        root = TrieNode()

        # global best word
        best_idx = 0

        for i in range(len(wordsContainer)):

            if len(wordsContainer[i]) < len(wordsContainer[best_idx]):
                best_idx = i

        # insert reversed words
        for i, word in enumerate(wordsContainer):

            rev = word[::-1]
            node = root

            # update root best
            if (len(word) < node.length or
               (len(word) == node.length and i < node.idx)):

                node.length = len(word)
                node.idx = i

            for ch in rev:

                if ch not in node.children:
                    node.children[ch] = TrieNode()

                node = node.children[ch]

                if (len(word) < node.length or
                   (len(word) == node.length and i < node.idx)):

                    node.length = len(word)
                    node.idx = i

        ans = []

        # process queries
        for word in wordsQuery:

            rev = word[::-1]
            node = root

            res = root.idx

            for ch in rev:

                if ch not in node.children:
                    break

                node = node.children[ch]
                res = node.idx

            ans.append(res)

        return ans
```
