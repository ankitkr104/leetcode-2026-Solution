# Problem Statement

You are given:
- an integer `mass` representing the initial mass of a planet
- an array `asteroids` where `asteroids[i]` is the mass of the i-th asteroid

The planet can collide with asteroids in any order.

Rules:

```text
If planet_mass >= asteroid_mass
    asteroid is destroyed
    planet_mass += asteroid_mass

Otherwise
    planet is destroyed
```

Return:

```text
true
```

if all asteroids can be destroyed, otherwise:

```text
false
```

---

## 🧪 Examples

### Example 1

```text
Input:
mass = 10
asteroids = [3,9,19,5,21]

Output:
true
```

Explanation:

```text
10 -> 13 -> 18 -> 27 -> 46 -> 67
```

All asteroids are destroyed.

---

### Example 2

```text
Input:
mass = 5
asteroids = [4,9,23,4]

Output:
false
```

Explanation:

```text
5 -> 9 -> 13 -> 22
```

Still cannot destroy asteroid 23.

---

## 📌 Constraints

```text
1 <= mass <= 10^5
1 <= asteroids.length <= 10^5
1 <= asteroids[i] <= 10^5
```

---

## 🧠 Approach

### 💡 Key Insight

To maximize the planet's growth:

```text
Always destroy the smallest asteroid first.
```

Why?

- Smaller asteroids are easier to absorb.
- Each absorption increases the planet's mass.
- This helps destroy larger asteroids later.

Therefore:

```text
Sort asteroids in ascending order.
```
