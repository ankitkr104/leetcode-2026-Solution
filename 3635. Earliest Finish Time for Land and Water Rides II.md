# Problem Statement

A tourist must take:

```text
Exactly one Land Ride
Exactly one Water Ride
```

in any order.

For each ride:

```text
Start at opening time or later
Finish = Start + Duration
```

After finishing the first ride:
- immediately take the second ride if open
- otherwise wait

Return the earliest possible finishing time.

---

## 🧪 Examples

### Example 1

```text
Input:
landStartTime = [2,8]
landDuration = [4,1]

waterStartTime = [6]
waterDuration = [3]

Output:
9
```

---

### Example 2

```text
Input:
landStartTime = [5]
landDuration = [3]

waterStartTime = [1]
waterDuration = [10]

Output:
14
```

---

## 📌 Constraints

```text
1 <= n,m <= 5 * 10^4

1 <= startTime <= 10^5
1 <= duration <= 10^5
```


---
## 💻 Solution (Python)

```python
from bisect import bisect_right

class Solution:
    def earliestFinishTime(
        self,
        landStartTime,
        landDuration,
        waterStartTime,
        waterDuration
    ):

        INF = float('inf')
        ans = INF

        # ---------- Land -> Water ----------
        water = sorted(zip(waterStartTime, waterDuration))

        ws = [x for x, _ in water]
        wd = [d for _, d in water]

        n = len(water)

        pref = [0] * n
        pref[0] = wd[0]

        for i in range(1, n):
            pref[i] = min(pref[i - 1], wd[i])

        suff = [0] * n
        suff[-1] = ws[-1] + wd[-1]

        for i in range(n - 2, -1, -1):
            suff[i] = min(
                suff[i + 1],
                ws[i] + wd[i]
            )

        for s, d in zip(landStartTime, landDuration):

            t = s + d

            pos = bisect_right(ws, t)

            if pos > 0:
                ans = min(
                    ans,
                    t + pref[pos - 1]
                )

            if pos < n:
                ans = min(
                    ans,
                    suff[pos]
                )

        # ---------- Water -> Land ----------
        land = sorted(zip(landStartTime, landDuration))

        ls = [x for x, _ in land]
        ld = [d for _, d in land]

        n = len(land)

        pref = [0] * n
        pref[0] = ld[0]

        for i in range(1, n):
            pref[i] = min(pref[i - 1], ld[i])

        suff = [0] * n
        suff[-1] = ls[-1] + ld[-1]

        for i in range(n - 2, -1, -1):
            suff[i] = min(
                suff[i + 1],
                ls[i] + ld[i]
            )

        for s, d in zip(waterStartTime, waterDuration):

            t = s + d

            pos = bisect_right(ls, t)

            if pos > 0:
                ans = min(
                    ans,
                    t + pref[pos - 1]
                )

            if pos < n:
                ans = min(
                    ans,
                    suff[pos]
                )

        return ans
```
