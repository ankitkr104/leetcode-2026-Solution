## 📘 Problem Statement

A tourist must take:

```text
Exactly one Land Ride
Exactly one Water Ride
```

in any order.

For each ride:

```text
Start at opening time or later
Finish = Start + Duration
```

After finishing the first ride:
- immediately take the second ride if open
- otherwise wait

Return the earliest possible finishing time.

---

## 🧪 Examples

### Example 1

```text
Input:
landStartTime = [2,8]
landDuration = [4,1]

waterStartTime = [6]
waterDuration = [3]

Output:
9
```

---

### Example 2

```text
Input:
landStartTime = [5]
landDuration = [3]

waterStartTime = [1]
waterDuration = [10]

Output:
14
```

---

## 📌 Constraints

```text
1 <= n,m <= 5 * 10^4

1 <= startTime <= 10^5
1 <= duration <= 10^5
```
