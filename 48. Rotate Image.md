# Problem Statement

You are given an `n x n` 2D matrix representing an image. Rotate the image by **90 degrees (clockwise)**.

⚠️ You must do this **in-place**, meaning:
- Modify the input matrix directly  
- ❌ Do NOT use another matrix  

---

## 🧪 Examples

### Example 1
Input:
matrix = [[1,2,3],
          [4,5,6],
          [7,8,9]]

Output:
[[7,4,1],
 [8,5,2],
 [9,6,3]]

---

### Example 2
Input:
matrix = [[5,1,9,11],
          [2,4,8,10],
          [13,3,6,7],
          [15,14,12,16]]

Output:
[[15,13,2,5],
 [14,3,4,1],
 [12,6,8,9],
 [16,7,10,11]]

---

## 📌 Constraints

- n == matrix.length == matrix[i].length  
- 1 ≤ n ≤ 20  
- -1000 ≤ matrix[i][j] ≤ 1000  

---

## 🧠 Approach

### 💡 Key Idea

To rotate the matrix **90° clockwise**, perform:

1. **Transpose the matrix**
2. **Reverse each row**

---

### 🔁 Step 1: Transpose

Convert rows → columns  
Swap:
matrix[i][j] ↔ matrix[j][i]

Example:

1 2 3 1 4 7
4 5 6 → 2 5 8
7 8 9 3 6 9


---

### 🔄 Step 2: Reverse Each Row


1 4 7 7 4 1
2 5 8 → 8 5 2
3 6 9 9 6 3


---

## 💻 Solution (Python)

```python
class Solution:
    def rotate(self, matrix):
        n = len(matrix)

        # Step 1: Transpose
        for i in range(n):
            for j in range(i, n):
                matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]

        # Step 2: Reverse each row
        for row in matrix:
            row.reverse()
