# Problem Statement

You are given an integer `n` representing the number of vertices in an **undirected graph**.

The graph has vertices numbered from

```text
0 to n-1
```

You are also given an array

```text
edges[i] = [u, v]
```

representing an undirected edge between vertices `u` and `v`.

Return the number of **complete connected components** in the graph.

A connected component is **complete** if every pair of vertices in that component has an edge between them.

---

## Example 1

```text
Input

n = 6

edges =
[[0,1],
 [0,2],
 [1,2],
 [3,4]]
```

Output

```text
3
```

### Explanation

Connected components are

```text
{0,1,2}

{3,4}

{5}
```

- `{0,1,2}` is complete (triangle).
- `{3,4}` is complete (one edge).
- `{5}` is also complete (single vertex).

Answer

```text
3
```

---

## Example 2

```text
Input

n = 6

edges =
[[0,1],
 [0,2],
 [1,2],
 [3,4],
 [3,5]]
```

Output

```text
1
```

### Explanation

Components

```text
{0,1,2}

{3,4,5}
```

The first component is complete.

The second is missing edge

```text
4-5
```

Therefore only one complete component exists.

---

## Constraints

```text
1 <= n <= 50

0 <= edges.length <= n(n-1)/2
```

---

# Python Solution

```python
from collections import defaultdict

class Solution:
    def countCompleteComponents(self, n, edges):

        graph = defaultdict(list)

        for u, v in edges:
            graph[u].append(v)
            graph[v].append(u)

        visited = [False] * n

        def dfs(node):

            stack = [node]

            vertices = 0
            degreeSum = 0

            visited[node] = True

            while stack:

                u = stack.pop()

                vertices += 1
                degreeSum += len(graph[u])

                for v in graph[u]:

                    if not visited[v]:

                        visited[v] = True
                        stack.append(v)

            return vertices, degreeSum // 2

        ans = 0

        for i in range(n):

            if not visited[i]:

                vertices, edgeCount = dfs(i)

                if edgeCount == vertices * (vertices - 1) // 2:

                    ans += 1

        return ans
```
