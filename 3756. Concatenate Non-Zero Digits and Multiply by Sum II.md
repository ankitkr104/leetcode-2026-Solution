# Problem Statement

You are given a string `s` consisting only of digits and a list of queries.

For each query `[l, r]`:

1. Extract the substring `s[l..r]`.
2. Form a new integer `x` by concatenating all **non-zero digits** of the substring in their original order.
   - If there are no non-zero digits, then `x = 0`.
3. Let `sum` be the sum of the digits of `x`.
4. Return the value:

```text
x × sum
```

Since the result can be very large, return every answer modulo

```text
10^9 + 7
```

---

## Example 1

```text
Input

s = "10203004"

queries =
[[0,7],
 [1,3],
 [4,6]]
```

Output

```text
[12340,4,9]
```

### Explanation

Query 1

```text
Substring

10203004

Non-zero digits

1234

x = 1234

Digit Sum

1+2+3+4 = 10

Answer

1234 × 10 = 12340
```

Query 2

```text
Substring

020

x = 2

Digit Sum = 2

Answer = 4
```

Query 3

```text
Substring

300

x = 3

Digit Sum = 3

Answer = 9
```

## Example 3

```text
Input

s = "9876543210"

queries =
[[0,9]]
```

Output

```text
444444137
```

### Explanation

```text
x = 987654321

Digit Sum

45

Product

987654321 × 45 = 44444444445

Modulo (10^9+7)

444444137
```

---

## Constraints

```text
1 <= s.length <= 100000

1 <= queries.length <= 100000

0 <= li <= ri < s.length
```
