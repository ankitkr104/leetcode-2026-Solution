# 2515. Shortest Distance to Target String in a Circular Array

## Problem Statement

You are given a **0-indexed circular string array** `words` and a string `target`.

A circular array means that the array's end connects to the beginning.

* The next element of `words[i]` is:

  ```text
  words[(i + 1) % n]
  ```
* The previous element of `words[i]` is:

  ```text
  words[(i - 1 + n) % n]
  ```

Starting from `startIndex`, you can move:

* Left (previous index)
* Right (next index)

Each move costs **1 step**.

---

## Goal

Return the **minimum number of steps** required to reach any index `i` such that:

```text
words[i] == target
```

If the target does not exist, return:

```text
-1
```

---

## Examples

### Example 1

```text
Input:
words = ["hello","i","am","leetcode","hello"]
target = "hello"
startIndex = 1

Output:
1
```

**Explanation:**

* Move left → index 0 → distance = 1 
* Other paths exist but are longer

---

### Example 2

```text
Input:
words = ["a","b","leetcode"]
target = "leetcode"
startIndex = 0

Output:
1
```

**Explanation:**

* Move left → index 2 → distance = 1 

---

### Example 3

```text
Input:
words = ["i","eat","leetcode"]
target = "ate"
startIndex = 0

Output:
-1
```

**Explanation:**

* Target does not exist

---

## Constraints

* `1 <= words.length <= 100`
* `1 <= words[i].length <= 100`
* `words[i]` and `target` consist of lowercase English letters
* `0 <= startIndex < words.length`

---

## Approach

For every index `i` where:

```text
words[i] == target
```

Compute circular distance:

```text
distance = min(|i - startIndex|, n - |i - startIndex|)
```

Take the minimum among all valid indices.

---

## Solution (Python)

```python
class Solution:
    def closestTarget(self, words, target, startIndex):
        n = len(words)
        ans = float('inf')
        
        for i in range(n):
            if words[i] == target:
                d = abs(i - startIndex)
                ans = min(ans, min(d, n - d))
        
        return ans if ans != float('inf') else -1
```
