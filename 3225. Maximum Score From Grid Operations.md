# Problem Statement

You are given a 2D matrix `grid` of size `n x n`. Initially, all cells of the grid are **white**.

In one operation, you can select any cell with indices `(i, j)` and color **black** all the cells in the `j-th` column starting from the **top row down to the i-th row**.

---

## 🎯 Objective

The **score** of the grid is defined as:

> The sum of all `grid[i][j]` such that:
> - The cell `(i, j)` is **white**, and  
> - It has at least one **horizontally adjacent black cell** (either left or right)

👉 Return the **maximum score** that can be achieved after performing any number of operations.

---

## 📌 Examples

### Example 1

```
Input:
grid = [
 [0,0,0,0,0],
 [0,0,3,0,0],
 [0,1,0,0,0],
 [5,0,0,3,0],
 [0,0,0,0,2]
]

Output:
11
```

---

### Example 2

```
Input:
grid = [
 [10,9,0,0,15],
 [7,1,0,8,0],
 [5,20,0,11,0],
 [0,0,0,1,2],
 [8,12,1,10,3]
]

Output:
94
```

---

## ⚙️ Constraints

- `1 <= n == grid.length <= 100`
- `n == grid[i].length`
- `0 <= grid[i][j] <= 10^9`
