# Problem Statement

You are given an integer array `nums`.

Your task is to count the number of ordered pairs of **non-empty**, **disjoint** subsequences

```text
(seq1, seq2)
```

such that:

- `seq1` and `seq2` share no common indices.
- The GCD of the elements in `seq1` equals the GCD of the elements in `seq2`.

Return the number of such pairs modulo

```text
10^9 + 7
```

---

## Example 1

```text
Input

nums = [1,2,3,4]
```

Output

```text
10
```

---

## Example 2

```text
Input

nums = [10,20,30]
```

Output

```text
2
```

---

## Example 3

```text
Input

nums = [1,1,1,1]
```

Output

```text
50
```

---

## Constraints

```text
1 <= nums.length <= 200

1 <= nums[i] <= 200
```

# Python3 Solution

```python
from math import gcd

class Solution:
    def subsequencePairCount(self, nums):

        MOD = 10 ** 9 + 7

        MAX = 200

        dp = [[0] * (MAX + 1) for _ in range(MAX + 1)]

        dp[0][0] = 1

        for x in nums:

            ndp = [row[:] for row in dp]

            for g1 in range(MAX + 1):

                for g2 in range(MAX + 1):

                    if dp[g1][g2] == 0:
                        continue

                    ng1 = x if g1 == 0 else gcd(g1, x)

                    ndp[ng1][g2] = (
                        ndp[ng1][g2] + dp[g1][g2]
                    ) % MOD

                    ng2 = x if g2 == 0 else gcd(g2, x)

                    ndp[g1][ng2] = (
                        ndp[g1][ng2] + dp[g1][g2]
                    ) % MOD

            dp = ndp

        ans = 0

        for g in range(1, MAX + 1):

            ans = (ans + dp[g][g]) % MOD

        return ans
```
