# Problem Statement

You are given an integer `side`, representing the edge length of a square with corners at:

- (0, 0)
- (0, side)
- (side, 0)
- (side, side)

You are also given:
- An integer `k`
- A 2D integer array `points`, where `points[i] = [xi, yi]` represents a point on the **boundary of the square**

---

## 🎯 Objective

Select **k points** from the given `points` such that:

> The **minimum Manhattan distance** between any pair of selected points is **maximized**

Return the maximum possible value of this minimum distance.

---

## 📌 Manhattan Distance

The Manhattan distance between two points `(x1, y1)` and `(x2, y2)` is:


|x1 - x2| + |y1 - y2|


---

## 🔍 Examples

### Example 1


Input:
side = 2
points = [[0,2],[2,0],[2,2],[0,0]]
k = 4

Output:
2


**Explanation:**  
All four points are selected, and the minimum Manhattan distance among them is `2`.

---

### Example 2


Input:
side = 2
points = [[0,0],[1,2],[2,0],[2,2],[2,1]]
k = 4

Output:
1


**Explanation:**  
One optimal selection is:

(0,0), (2,0), (2,2), (2,1)

The minimum Manhattan distance is `1`.

---

### Example 3


Input:
side = 2
points = [[0,0],[0,1],[0,2],[1,2],[2,0],[2,2],[2,1]]
k = 5

Output:
1


**Explanation:**  
One optimal selection is:

(0,0), (0,1), (0,2), (1,2), (2,2)


---

## ⚙️ Constraints

- `1 <= side <= 10^9`
- `4 <= points.length <= min(4 * side, 15 * 10^3)`
- `points[i] = [xi, yi]`
- Each point lies on the **boundary of the square**
- All points are **unique**
- `4 <= k <= min(25, points.length)`
