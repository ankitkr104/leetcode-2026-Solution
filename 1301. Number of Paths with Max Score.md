# Problem Statement

You are given an `n × n` board of characters.

The board contains:

- `E` → Start (top-left)
- `S` → Destination (bottom-right)
- Digits `'1'` to `'9'`
- `X` → Obstacle

You start from

```text
S (bottom-right)
```

and want to reach

```text
E (top-left)
```

You may move only:

- Up
- Left
- Up-Left (Diagonal)

You cannot move onto an obstacle.

For every valid path:

- Sum all numeric cells visited.
- Ignore `S` and `E`.

Return:

```text
[maxScore, numberOfPaths]
```

where

- `maxScore` is the maximum obtainable score.
- `numberOfPaths` is the number of paths giving that score modulo

```text
10^9 + 7
```

If no valid path exists, return

```text
[0,0]
```

---

# Example 1

```text
Input

board =
["E23",
 "2X2",
 "12S"]
```

Output

```text
[7,1]
```

```text
2 ≤ n ≤ 100
```

# Python Solution

```python
class Solution:
    def pathsWithMaxScore(self, board):

        MOD = 10 ** 9 + 7

        n = len(board)

        score = [[-1] * n for _ in range(n)]
        ways = [[0] * n for _ in range(n)]

        score[n - 1][n - 1] = 0
        ways[n - 1][n - 1] = 1

        for i in range(n - 1, -1, -1):

            for j in range(n - 1, -1, -1):

                if board[i][j] == "X":
                    continue

                if i == n - 1 and j == n - 1:
                    continue

                best = -1
                cnt = 0

                for x, y in (
                    (i + 1, j),
                    (i, j + 1),
                    (i + 1, j + 1),
                ):

                    if x >= n or y >= n:
                        continue

                    if score[x][y] == -1:
                        continue

                    if score[x][y] > best:

                        best = score[x][y]
                        cnt = ways[x][y]

                    elif score[x][y] == best:

                        cnt = (cnt + ways[x][y]) % MOD

                if best == -1:
                    continue

                score[i][j] = best

                if board[i][j].isdigit():

                    score[i][j] += int(board[i][j])

                ways[i][j] = cnt % MOD

        if ways[0][0] == 0:

            return [0, 0]

        return [score[0][0], ways[0][0]]
```
