# Problem Statement

You are given an integer array `nums` of length `n`, where `nums` is a permutation of the integers in the range

```text
[1, n]
```

A **XOR triplet** is defined as

```text
nums[i] XOR nums[j] XOR nums[k]
```

where

```text
i <= j <= k
```

Return the number of **distinct XOR values** that can be obtained from all possible triplets.

---

## Example 1

```text
Input

nums = [1,2]
```

Output

```text
2
```

### Explanation

Possible XOR values are

```text
1 XOR 1 XOR 1 = 1

1 XOR 1 XOR 2 = 2

1 XOR 2 XOR 2 = 1

2 XOR 2 XOR 2 = 2
```

Unique values

```text
{1,2}
```

Answer

```text
2
```

---

## Example 2

```text
Input

nums = [3,1,2]
```

Output

```text
4
```

### Explanation

Some triplets are

```text
3 XOR 3 XOR 3 = 3

3 XOR 3 XOR 1 = 1

3 XOR 3 XOR 2 = 2

3 XOR 1 XOR 2 = 0
```

Unique values

```text
{0,1,2,3}
```

Answer

```text
4
```

---

## Constraints

```text
1 <= n <= 100000

nums is a permutation of [1, n]
```
