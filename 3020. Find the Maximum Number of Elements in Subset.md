## Problem Statement

You are given an array of positive integers `nums`.

You need to select a subset of `nums` that can be rearranged into the following form:

```text
[x, x², x⁴, ..., x^(k/2), x^k, x^(k/2), ..., x⁴, x², x]
```

where:

- `k` is any non-negative power of 2.
- Every element must exist in the chosen subset.

Return the maximum possible size of such a subset.

---

## Example 1

```text
Input:
nums = [5,4,1,2,2]

Output:
3
```

Explanation:

Choose

```text
{2,2,4}
```

Arrange as

```text
[2,4,2]
```

Since

```text
2² = 4
```

Answer:

```text
3
```

---

## Example 2

```text
Input:
nums = [1,3,2,4]

Output:
1
```

Explanation:

No chain can be formed.

Choose any single element.

Answer:

```text
1
```

---

## Constraints

```text
2 <= nums.length <= 100000

1 <= nums[i] <= 10^9
```
