## Problem Statement

You are given a **Directed Acyclic Graph (DAG)** with `n` nodes numbered from `0` to `n-1`.

Each directed edge is represented as:

```text
[u, v, cost]
```

meaning there is a one-way edge from node `u` to node `v` with recovery cost `cost`.

You are also given:

- `online[i]` indicating whether node `i` is online.
- An integer `k`.

Nodes

```text
0
```

and

```text
n-1
```

are always online.

A path from

```text
0 → n-1
```

is **valid** if:

- Every intermediate node is online.
- Total edge cost is at most

```text
k
```

The **score** of a path is defined as

```text
minimum edge cost on that path.
```

Return the **maximum possible path score**.

If no valid path exists, return

```text
-1
```

---

# Example 1

```text
Input

edges =
[[0,1,5],
 [1,3,10],
 [0,2,3],
 [2,3,4]]

online =
[true,true,true,true]

k = 10
```

Output

```text
3
```

Explanation

Possible paths:

```text
0 → 1 → 3

Cost

15

> 10

Invalid
```

```text
0 → 2 → 3

Cost

7

Valid

Minimum edge

min(3,4)=3
```

Answer

```text
3
```

---

# Example 2

```text
Input

edges =
[[0,1,7],
 [1,4,5],
 [0,2,6],
 [2,3,6],
 [3,4,2],
 [2,4,6]]

online =
[true,true,true,false,true]

k = 12
```

Output

```text
6
```

Explanation

Node

```text
3
```

is offline.

So

```text
0→2→3→4
```

cannot be used.

Remaining paths

```text
0→1→4

Score = 5
```

```text
0→2→4

Score = 6
```

Answer

```text
6
```

---

# Constraints

```text
2 ≤ n ≤ 50000

0 ≤ m ≤ 100000

Graph is a DAG

0 ≤ cost ≤ 10^9

0 ≤ k ≤ 5×10^13
```
