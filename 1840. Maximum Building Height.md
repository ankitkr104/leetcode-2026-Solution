# Problem Statement

You want to build `n` buildings numbered from:

```text
1 → n
```

subject to the following rules:

### Building Constraints

1. Heights must be non-negative integers.

2. Building `1` must have height:

```text
0
```

3. Adjacent buildings can differ by at most:

```text
1
```

```text
|height[i] - height[i-1]| ≤ 1
```

4. Some buildings have additional restrictions:

```text
[id, maxHeight]
```

meaning

```text
height[id] ≤ maxHeight
```

Return the maximum possible height of the tallest building.

---

n = 5
restrictions = [[2,1],[4,1]]

Output:
2
```

```

Maximum height:

```text
2
```

## 🧠 Key Observation

Suppose we have two restricted buildings:

```text
(id1, h1)
(id2, h2)
```

Because adjacent buildings change by at most 1:

```text
h2 ≤ h1 + (id2-id1)
h1 ≤ h2 + (id2-id1)
```

Therefore restrictions must be tightened first.

---

## Step 1: Add Boundary Restrictions

Building 1:

```text
(1,0)
```

Building n:

```text
(n,n-1)
```

because without any restriction the largest possible height at n is:

```text
n-1
```

---

## Step 2: Forward Pass

Limit every restriction using the previous one:

```python
h[i] = min(
    h[i],
    h[i-1] + distance
)
```

---

## Step 3: Backward Pass

Limit every restriction using the next one:

```python
h[i] = min(
    h[i],
    h[i+1] + distance
)
```

After these two passes all restrictions become feasible.

---

## Step 4: Find Maximum Peak

Between two neighboring restrictions:

```text
(id1,h1)
(id2,h2)
```

Let:

```text
d = id2-id1
```

The highest achievable peak between them is:

```text
(h1 + h2 + d) // 2
```

Why?

Because height can increase by 1 per step and later decrease by 1 per step.

The tallest point is where both slopes meet.

---

## 💻 Solution (Python)

```python
class Solution:
    def maxBuilding(self, n, restrictions):

        restrictions.append([1, 0])

        restrictions.append([n, n - 1])

        restrictions.sort()

        m = len(restrictions)

        # Forward pass
        for i in range(1, m):

            dist = restrictions[i][0] - restrictions[i - 1][0]

            restrictions[i][1] = min(
                restrictions[i][1],
                restrictions[i - 1][1] + dist
            )

        # Backward pass
        for i in range(m - 2, -1, -1):

            dist = restrictions[i + 1][0] - restrictions[i][0]

            restrictions[i][1] = min(
                restrictions[i][1],
                restrictions[i + 1][1] + dist
            )

        ans = 0

        for i in range(1, m):

            x1, h1 = restrictions[i - 1]
            x2, h2 = restrictions[i]

            dist = x2 - x1

            peak = (h1 + h2 + dist) // 2

            ans = max(ans, peak)

        return ans
```
