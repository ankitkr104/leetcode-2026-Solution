# Problem Statement

You are given a string `s` consisting of:

```text
Lowercase English letters
*
#
%
```

Build a new string `result` by processing characters from left to right.

### Operations

#### Letter

```text
Append the letter to result
```

#### '*'

```text
Remove the last character from result
(if it exists)
```

#### '#'

```text
Duplicate the current result

result = result + result
```

#### '%'

```text
Reverse the current result
```

Return the final string after processing all characters.

---

## 🧪 Example 1

```text
Input:
s = "a#b%*"

Output:
"ba"
```

### Explanation

```text
"a"      -> append a
"aa"     -> duplicate
"aab"    -> append b
"baa"    -> reverse
"ba"     -> remove last character
```

Final result:

```text
"ba"
```

---

## 🧪 Example 2

```text
Input:
s = "z*#"

Output:
""
```

### Explanation

```text
"z" -> append z
""  -> remove z
""  -> duplicate empty string
```

Final result:

```text
""
```

---

## 📌 Constraints

```text
1 <= s.length <= 20

s consists only of:

'a' - 'z'
*
#
%
```
