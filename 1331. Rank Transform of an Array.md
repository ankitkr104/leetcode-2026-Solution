# Problem Statement

You are given an integer array `arr`.

Replace every element with its **rank**.

The rank follows these rules:

- Rank starts from **1**.
- Larger numbers have larger ranks.
- Equal numbers receive the **same rank**.
- Ranks should be as small as possible.

Return the transformed array.

---

## Example 1

```text
Input

arr = [40,10,20,30]
```

Output

```text
[4,1,2,3]
```

### Explanation

Sorted unique values:

```text
10 20 30 40
```

Ranks:

```text
10 → 1

20 → 2

30 → 3

40 → 4
```

Therefore

```text
[40,10,20,30]

↓

[4,1,2,3]
```

---

## Example 2

```text
Input

arr = [100,100,100]
```

Output

```text
[1,1,1]
```

Explanation

All values are identical, so they share the same rank.

Input
