# Problem Statement

You are given:

```text
nums
```

and an integer:

```text
pivot
```

Rearrange the array such that:

1. All elements smaller than `pivot` appear first.
2. All elements equal to `pivot` appear next.
3. All elements greater than `pivot` appear last.
4. The relative order of elements smaller than pivot is preserved.
5. The relative order of elements greater than pivot is preserved.

Return the rearranged array.

---

## 🧪 Examples

### Example 1

```text
Input:
nums = [9,12,5,10,14,3,10]
pivot = 10

Output:
[9,5,3,10,10,12,14]
```

Explanation:

```text
Less than 10:
[9,5,3]

Equal to 10:
[10,10]

Greater than 10:
[12,14]
```

Combined:

```text
[9,5,3,10,10,12,14]
```

---

### Example 2

```text
Input:
nums = [-3,4,3,2]
pivot = 2

Output:
[-3,2,4,3]
```

---

## 📌 Constraints

```text
1 <= nums.length <= 10^5

-10^6 <= nums[i] <= 10^6

pivot is guaranteed to exist in nums
```

---

## 💻 Solution (Python)

```python
class Solution:
    def pivotArray(self, nums, pivot):

        less = []
        equal = []
        greater = []

        for num in nums:

            if num < pivot:
                less.append(num)

            elif num == pivot:
                equal.append(num)

            else:
                greater.append(num)

        return less + equal + greater
```
