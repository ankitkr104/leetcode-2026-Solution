# Problem Statement

You are given:

```text
nums
```

of length `n` and an integer:

```text
k
```

You must choose exactly:

```text
k non-empty subarrays
```

Subarrays:
- may overlap
- may be identical

The value of a subarray:

```text
max(subarray) - min(subarray)
```

Return the maximum possible total value.

---

## 🧪 Examples

### Example 1

```text
Input:
nums = [1,3,2]
k = 2

Output:
4
```

Explanation:

```text
[1,3]     → 3-1 = 2
[1,3,2]   → 3-1 = 2

Total = 4
```

---

### Example 2

```text
Input:
nums = [4,2,5,1]
k = 3

Output:
12
```

Explanation:

```text
[4,2,5,1] → 5-1 = 4
[4,2,5,1] → 5-1 = 4
[5,1]     → 5-1 = 4

Total = 12
```

---

## 📌 Constraints

```text
1 <= n == nums.length <= 5 * 10^4

0 <= nums[i] <= 10^9

1 <= k <= 10^5
```
