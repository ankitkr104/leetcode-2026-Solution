# Problem Statement

You are given a 2D integer array `descriptions` where:

```text
descriptions[i] = [parent, child, isLeft]
```

indicates:

- `parent` is the parent node of `child`
- If `isLeft = 1`, then `child` is the left child
- If `isLeft = 0`, then `child` is the right child

Construct the binary tree and return its root.

It is guaranteed that the resulting binary tree is valid.

---

## 🧪 Examples

### Example 1

```text
Input:
descriptions =
[[20,15,1],
 [20,17,0],
 [50,20,1],
 [50,80,0],
 [80,19,1]]

Output:
[50,20,80,15,17,19]
```

Explanation:

```text
        50
       /  \
     20    80
    / \    /
  15  17 19
```

Root = 50

---

### Example 2

```text
Input:
descriptions =
[[1,2,1],
 [2,3,0],
 [3,4,1]]

Output:
[1,2,null,null,3,4]
```

Explanation:

```text
    1
   /
  2
   \
    3
   /
  4
```

Root = 1

---

## 📌 Constraints

```text
1 <= descriptions.length <= 10^4

descriptions[i].length == 3

1 <= parenti, childi <= 10^5

0 <= isLefti <= 1

The binary tree described is valid.
```
