## Problem Statement

You are given:

- An integer `n` representing the number of nodes labeled from `0` to `n-1`.
- An integer array `nums` of length `n`.
- An integer `maxDiff`.

An **undirected** edge exists between nodes `i` and `j` if

```text
|nums[i] - nums[j]| ≤ maxDiff
```

The graph is **unweighted**.

You are also given several queries.

For each query

```text
[u, v]
```

return the **minimum number of edges** in a path between `u` and `v`.

If no path exists, return

```text
-1
```

---

## Example 1

```text
Input

n = 5

nums = [1,8,3,4,2]

maxDiff = 3

queries =
[[0,3],
 [2,4]]
```

Output

```text
[1,1]
```

Explanation

```text
0 → 3

distance = 1
```

```text
2 → 4

distance = 1
```

queries =
[[0,1],
 [0,2],

 [4,3]]
```

[1,2,-1,1]
```

## Example 3

```text
Input

nums = [3,6,1]

maxDiff = 1

queries =
[[0,0],
 [0,1],
 [1,2]]
```

Output

```text
[0,-1,-1]
```

---

## Constraints

```text
1 ≤ n ≤ 100000

1 ≤ queries ≤ 100000

0 ≤ nums[i] ≤ 100000
```
