## Problem Statement

There are `n` houses evenly lined up on a street, and each house is painted with a color.

You are given a **0-indexed integer array** `colors` of length `n`, where:
- `colors[i]` represents the color of the `i-th` house.

Return the **maximum distance** between two houses with **different colors**.

The distance between two houses `i` and `j` is:


abs(i - j)


---

## Examples

### Example 1


Input: colors = [1,1,1,6,1,1,1]
Output: 3


**Explanation:**
- House `0` (color `1`) and house `3` (color `6`)
- Distance = `abs(0 - 3) = 3`

---

### Example 2


Input: colors = [1,8,3,8,3]
Output: 4


**Explanation:**
- House `0` (color `1`) and house `4` (color `3`)
- Distance = `abs(0 - 4) = 4`

---

### Example 3


Input: colors = [0,1]
Output: 1


---

## Constraints

- `n == colors.length`
- `2 <= n <= 100`
- `0 <= colors[i] <= 100`
- At least two houses have different colors

---

## 💡 Solution
---

### Code

```python id="v6t9xp"
class Solution:
    def maxDistance(self, colors):
        n = len(colors)
        
        max_dist = 0
        
        for i in range(n):
            if colors[i] != colors[0]:
                max_dist = max(max_dist, i)
            if colors[i] != colors[-1]:
                max_dist = max(max_dist, n - 1 - i)
                
        return max_dist
