# Problem Statement

In a linked list of even length `n`, the twin of the `i-th` node is:

```text
(n - 1 - i)-th node
```

The **twin sum** is defined as:

```text
node[i] + node[n-1-i]
```

Return the maximum twin sum of the linked list.

---

## 🧪 Examples

### Example 1

```text
Input:
head = [5,4,2,1]

Output:
6
```

### Explanation

```text
Twin pairs:

5 + 1 = 6
4 + 2 = 6

Maximum = 6
```

---

### Example 2

```text
Input:
head = [4,2,2,3]

Output:
7
```

### Explanation

```text
Twin pairs:

4 + 3 = 7
2 + 2 = 4

Maximum = 7
```

---

### Example 3

```text
Input:
head = [1,100000]

Output:
100001
```

### Explanation

```text
Only one twin pair:

1 + 100000 = 100001
```

---

## 📌 Constraints

```text
2 <= Number of Nodes <= 10^5

Number of Nodes is even.

1 <= Node.val <= 10^5
```

---
## 💻 Solution (Python)

```python
class Solution:
    def pairSum(self, head):

        nums = []

        while head:
            nums.append(head.val)
            head = head.next

        left = 0
        right = len(nums) - 1

        ans = 0

        while left < right:

            ans = max(
                ans,
                nums[left] + nums[right]
            )

            left += 1
            right -= 1

        return ans
```
