# Problem Statement

You are given an integer array `nums` of length `n`.

Construct an array `prefixGcd` as follows:

For every index `i`:

- Let

```text
mxi = max(nums[0], nums[1], ..., nums[i])
```

- Then

```text
prefixGcd[i] = gcd(nums[i], mxi)
```

After constructing `prefixGcd`:

1. Sort it in **non-decreasing order**.
2. Pair the **smallest** unpaired element with the **largest** unpaired element.
3. Continue until no more pairs can be formed.
4. For every pair, compute the **GCD** of the two numbers.
5. If the array has an odd length, ignore the middle element.

Return the **sum of the GCDs** of all formed pairs.

---

## Example 1

```text
Input

nums = [2,6,4]
```

Output

```text
2
```

### Explanation

Construct `prefixGcd`

| i | nums[i] | Maximum so far | prefixGcd |
|---|---------|----------------|-----------|
|0|2|2|2|
|1|6|6|6|
|2|4|6|2|

```text
prefixGcd = [2,6,2]
```

Sort

```text
[2,2,6]
```

Pair

```text
(2,6)

gcd = 2
```

The middle element remains unused.

Answer

```text
2
```

## Example 2

```text
Input

nums = [3,6,2,8]
```

Output

```text
5
```

### Explanation

Construct

```text
prefixGcd

[3,6,2,8]
```

Sort

```text
[2,3,6,8]
```

Pairs

```text
(2,8)

gcd = 2
```

```text
(3,6)

gcd = 3
```

Total

```text
2 + 3 = 5
```

---

## Constraints

```text
1 <= n <= 100000

1 <= nums[i] <= 10^9
```

---

# Python Solution

```python
class Solution:
    def gcd(self, a, b):
        while b:
            a, b = b, a % b
        return a

    def gcdSum(self, nums):
        prefix = []
        mx = 0

        for x in nums:
            if x > mx:
                mx = x
            prefix.append(self.gcd(x, mx))

        prefix.sort()

        ans = 0
        i, j = 0, len(prefix) - 1

        while i < j:
            ans += self.gcd(prefix[i], prefix[j])
            i += 1
            j -= 1

        return ans
```
