## 🧾 Problem Statement

You are given a 2D integer grid of size `m x n` and an integer `x`.

In one operation, you can:
- Add `x` to any element, or
- Subtract `x` from any element.

A **uni-value grid** is a grid where all elements are equal.

👉 Return the **minimum number of operations** to make the grid uni-value.  
👉 If it is **not possible**, return `-1`.

---

## 📌 Examples

### Example 1
```
Input: grid = [[2,4],[6,8]], x = 2
Output: 4
```

**Explanation:**
Make all elements equal to 4:
- 2 → 4 → +2 → 1 operation  
- 6 → 4 → -2 → 1 operation  
- 8 → 4 → -4 → 2 operations  

Total = **4 operations**

---

### Example 2
```
Input: grid = [[1,5],[2,3]], x = 1
Output: 5
```

**Explanation:**
Make all elements equal to 3.

---

### Example 3
```
Input: grid = [[1,2],[3,4]], x = 2
Output: -1
```

**Explanation:**
Not all elements can reach the same value using steps of `x`.

---

## ⚙️ Constraints

- `m == grid.length`
- `n == grid[i].length`
- `1 <= m, n <= 10^5`
- `1 <= m * n <= 10^5`
- `1 <= x, grid[i][j] <= 10^4`

---

## 🧠 Approach

### 🔑 Key Observations

1. Since we can only add/subtract `x`,  
   all elements must have the **same remainder when divided by `x`**.

   👉 If not, return `-1`.

2. Convert the grid into a **1D array**.

3. Sort the array.

4. Choose the **median** as the target value:
   - Minimizes total number of operations.

5. Count operations:
   ```
   operations += abs(num - median) // x
   ```

---

## 🚀 Algorithm

1. Flatten grid into array `arr`
2. Check if all elements have same `num % x`
3. Sort array
4. Pick median
5. Calculate total operations

---

## 💻 Solution (Python)

```python
class Solution:
    def minOperations(self, grid, x):
        # Step 1: Flatten grid
        arr = [num for row in grid for num in row]
        
        # Step 2: Check feasibility
        remainder = arr[0] % x
        for num in arr:
            if num % x != remainder:
                return -1
        
        # Step 3: Sort and find median
        arr.sort()
        median = arr[len(arr) // 2]
        
        # Step 4: Count operations
        operations = 0
        for num in arr:
            operations += abs(num - median) // x
        
        return operations
```
