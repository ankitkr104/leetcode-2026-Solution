# 🔁 796. Rotate String

**Difficulty:** Easy  
**Topics:** String  
**Status:** ✅ Solved  

---

## 📘 Problem Statement

Given two strings `s` and `goal`, return `true` if and only if `s` can become `goal` after some number of **shifts**.

A shift operation consists of moving the **leftmost character** of `s` to the **rightmost position**.

### Example:
s = "abcde" → "bcdea" (after 1 shift)

---

## 🧪 Examples

### Example 1
Input:  s = "abcde", goal = "cdeab"  
Output: true  

### Example 2
Input:  s = "abcde", goal = "abced"  
Output: false  

---

## 📌 Constraints

- 1 ≤ s.length, goal.length ≤ 100  
- s and goal consist of lowercase English letters  

---

## 🧠 Approach

### 💡 Key Idea

If `s` can be rotated to become `goal`, then:

goal must be a substring of (s + s)

### 🔍 Why?

All possible rotations of `s` are present inside `s + s`.

Example:
s = "abcde"  
s + s = "abcdeabcde"  

Possible rotations:
abcde, bcdea, cdeab, deabc, eabcd  

---

## ⚠️ Important Check

- Length of `s` and `goal` must be equal

---

## 💻 Solution (Python)

```python
class Solution:
    def rotateString(self, s, goal):
        return len(s) == len(goal) and goal in (s + s)
