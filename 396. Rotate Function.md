## 📘 Problem Description

You are given an integer array `nums` of length `n`.

Let `arr_k` be the array obtained by **rotating `nums` clockwise by `k` positions**.

We define a rotation function:

F(k) = 0 * arr_k[0] + 1 * arr_k[1] + ... + (n - 1) * arr_k[n - 1]

### 🎯 Goal
Return the **maximum value** among all possible rotations:

max(F(0), F(1), ..., F(n-1))

---

## 🧪 Examples

### Example 1
Input:
nums = [4,3,2,6]

Output:
26

Explanation:
F(0) = 0*4 + 1*3 + 2*2 + 3*6 = 25  
F(1) = 0*6 + 1*4 + 2*3 + 3*2 = 16  
F(2) = 0*2 + 1*6 + 2*4 + 3*3 = 23  
F(3) = 0*3 + 1*2 + 2*6 + 3*4 = 26  

✅ Maximum = 26

---

### Example 2
Input:
nums = [100]

Output:
0

---

## 🧠 Approach

### ❌ Brute Force
- Rotate array `n` times
- Compute `F(k)` each time  
- Time Complexity: **O(n²)** → Not efficient

---

### ✅ Optimized Approach (Key Insight)

Let:
- `total_sum = sum(nums)`
- `F(0) = sum(i * nums[i])`

Then:

F(k) = F(k-1) + total_sum - n * nums[n - k]

---

## 🔁 Recurrence Relation

F(k) = F(k-1) + sum(nums) - n * nums[n - k]
