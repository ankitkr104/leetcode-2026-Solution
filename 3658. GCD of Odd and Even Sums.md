# Problem Statement

You are given an integer `n`.

Compute the following two values:

- **sumOdd** = sum of the first `n` positive odd numbers.
- **sumEven** = sum of the first `n` positive even numbers.

Return the **Greatest Common Divisor (GCD)** of

```text
sumOdd

and

sumEven
```

---

## Example 1

```text
Input

n = 4
```

Output

```text
4
```

### Explanation

First 4 odd numbers

```text
1 + 3 + 5 + 7 = 16
```

First 4 even numbers

```text
2 + 4 + 6 + 8 = 20
```

Therefore

```text
GCD(16,20)=4
```

---

## Example 2

```

```

First 5 even numbers

```text

```

```text

```

---

## Constraints

```text
1 <= n <= 1000
```
