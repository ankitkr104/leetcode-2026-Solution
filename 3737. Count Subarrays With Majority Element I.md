## Problem Statement

You are given an integer array `nums` and an integer `target`.

Return the number of subarrays in which `target` is the **majority element**.

A majority element is one that appears **strictly more than half** of the total elements in the subarray.

---

## Example 1

```text
Input:
nums = [1,2,2,3]
target = 2

Output:
5
```

Valid subarrays:

```text
[2]
[2]
[2,2]
[1,2,2]
[2,2,3]
```

Answer:

```text
5
```

---

## Example 2

```text
Input:
nums = [1,1,1,1]
target = 1

Output:
10
```

Every subarray has 1 as majority.

---

## Example 3

```text
Input:
nums = [1,2,3]
target = 4

Output:
0
```

Target never appears.

---

## Constraints

```text
1 <= n <= 1000

1 <= nums[i] <= 10^9
```
