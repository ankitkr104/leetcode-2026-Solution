## Problem Statement

You are given an integer array `nums`.

A **mirror pair** is a pair of indices `(i, j)` such that:

* `0 <= i < j < nums.length`, and
* `reverse(nums[i]) == nums[j]`

Where:

```text id="8g2l3k"
reverse(x) = integer formed by reversing digits of x
```

 Note: Leading zeros are removed after reversing
Example:

```text id="x3w9qa"
reverse(120) = 21
```

---

## Goal

Return the **minimum absolute distance** between indices of any mirror pair:

```text id="a1pz8n"
distance = |i - j|
```

If no mirror pair exists, return:

```text id="z9kq2m"
-1
```

---

## Examples

### Example 1

```text id="p3s8rf"
Input:
nums = [12,21,45,33,54]

Output:
1
```

**Explanation:**

* (0,1): reverse(12) = 21 → distance = 1
* (2,4): reverse(45) = 54 → distance = 2

Minimum = **1**

---

### Example 2

```text id="t6g1vb"
Input:
nums = [120,21]

Output:
1
```

**Explanation:**

* (0,1): reverse(120) = 21 → distance = 1

---

### Example 3

```text id="f9d2xp"
Input:
nums = [21,120]

Output:
-1
```

**Explanation:**

* No mirror pair exists

---

## Constraints

* `1 <= nums.length <= 10^5`
* `1 <= nums[i] <= 10^9`

---

## ✅ Solution (Python)

```python id="k9r2bz"
class Solution:
    def minMirrorPairDistance(self, nums):
        seen = {}
        ans = float('inf')
        
        for i, x in enumerate(nums):
            rev = int(str(x)[::-1])
            
            # check if current matches reverse of previous
            if x in seen:
                ans = min(ans, i - seen[x])
            
            # store reversed value
            seen[rev] = i
        
        return ans if ans != float('inf') else -1
```

---

## Explanation

* Traverse the array from left to right
* For each element:

  * Compute its reversed value
  * Check if the current number exists in the hashmap

    * If yes → we found a mirror pair
* Store the reversed value as a key with its index
