# Problem Statement

You are given a **0-indexed integer array** `nums`.

Define an array `arr` such that:


arr[i] = sum of |i - j| for all j where nums[j] == nums[i] and j != i


If no such index `j` exists, then:


arr[i] = 0


---

### Task

Return the array `arr`.

---

## Examples

### Example 1


Input: nums = [1,3,1,1,2]
Output: [5,0,3,4,0]


**Explanation:**

- `i = 0` → same values at indices `2, 3`  
  → `|0-2| + |0-3| = 5`

- `i = 1` → no same value → `0`

- `i = 2` → indices `0, 3`  
  → `|2-0| + |2-3| = 3`

- `i = 3` → indices `0, 2`  
  → `|3-0| + |3-2| = 4`

- `i = 4` → no same value → `0`

---

### Example 2


Input: nums = [0,5,3]
Output: [0,0,0]


---

## ⚙️ Constraints

- `1 <= nums.length <= 10^5`
- `0 <= nums[i] <= 10^9`

---

## Solution
### 💻 Code

```python
from collections import defaultdict

class Solution:
    def distance(self, nums):
        groups = defaultdict(list)
        
        # Step 1: Group indices
        for i, num in enumerate(nums):
            groups[num].append(i)
        
        res = [0] * len(nums)
        
        # Step 2: Process each group
        for indices in groups.values():
            n = len(indices)
            
            # Prefix sum of indices
            prefix = [0] * n
            prefix[0] = indices[0]
            
            for i in range(1, n):
                prefix[i] = prefix[i - 1] + indices[i]
            
            # Step 3: Calculate distances
            for i in range(n):
                idx = indices[i]
                
                # Left side contribution
                left = i * idx - (prefix[i - 1] if i > 0 else 0)
                
                # Right side contribution
                right = (prefix[n - 1] - prefix[i]) - (n - i - 1) * idx
                
                res[idx] = left + right
        
        return res
