# Problem Statement

You are given:

```text
nums
```

and an integer:

```text
k
```

You must select exactly:

```text
k DISTINCT non-empty subarrays
```

A subarray value is:

```text
max(subarray) - min(subarray)
```

Return the maximum possible sum of values of the chosen subarrays.

---

## 🧪 Examples

### Example 1

```text
Input:
nums = [1,3,2]
k = 2

Output:
4
```

Explanation:

```text
[1,3]   → 2
[1,3,2] → 2

Total = 4
```

---

### Example 2

```text
Input:
nums = [4,2,5,1]
k = 3

Output:
12
```

Explanation:

```text
[4,2,5,1] → 4
[2,5,1]   → 4
[5,1]     → 4

Total = 12
```

---

## 📌 Constraints

```text
1 <= n <= 5 * 10^4

0 <= nums[i] <= 10^9

1 <= k <= min(10^5, n(n+1)/2)
```
