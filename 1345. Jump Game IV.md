# Problem Statement

You are given an integer array `arr`.

You start at index `0`.

In one step, you can jump from index `i` to:

```text
i + 1   (if within bounds)
i - 1   (if within bounds)
j where arr[i] == arr[j] and i != j
```

Return the **minimum number of steps** required to reach the last index.

⚠️ You cannot jump outside the array.

---

## 🧪 Examples

### Example 1
```text
Input:
arr = [100,-23,-23,404,100,23,23,23,3,404]

Output:
3
```

Explanation:
```text
0 → 4 → 3 → 9
```

---

### Example 2
```text
Input:
arr = [7]

Output:
0
```

Explanation:
```text
Already at last index
```

---

### Example 3
```text
Input:
arr = [7,6,9,6,9,6,9,7]

Output:
1
```

Explanation:
```text
0 → 7
```

---

## 📌 Constraints

```text
1 <= arr.length <= 5 * 10^4
-10^8 <= arr[i] <= 10^8
```

---

## 🧠 Approach

### 💡 Key Insight

This is a **shortest path problem**.

Each index behaves like a graph node.

Possible edges:
- left neighbor
- right neighbor
- all indices with same value

Use:
```text
BFS
```

because every jump costs exactly `1`.

---

## 🔥 Optimization Trick

If we repeatedly visit all same-value indices,
time complexity becomes huge.

Example:
```text
[7,7,7,7,7,7...]
```

To avoid TLE:
✅ After processing indices of one value,
remove them from the hashmap.

---

## 🔁 BFS Strategy

1. Store:
```text
value → list of indices
```

2. BFS from index `0`

3. Explore:
- `i - 1`
- `i + 1`
- all same-value indices

4. Use visited set

5. Delete processed value group

---
## 💻 Solution (Python)

```python
from collections import defaultdict, deque

class Solution:
    def minJumps(self, arr):

        n = len(arr)

        if n == 1:
            return 0

        graph = defaultdict(list)

        # value -> indices
        for i, val in enumerate(arr):
            graph[val].append(i)

        q = deque([0])
        visited = set([0])

        steps = 0

        while q:

            for _ in range(len(q)):

                i = q.popleft()

                if i == n - 1:
                    return steps

                neighbors = []

                # adjacent jumps
                if i - 1 >= 0:
                    neighbors.append(i - 1)

                if i + 1 < n:
                    neighbors.append(i + 1)

                # same value jumps
                neighbors.extend(graph[arr[i]])

                for nxt in neighbors:

                    if nxt not in visited:
                        visited.add(nxt)
                        q.append(nxt)

                # prevent repeated processing
                graph[arr[i]].clear()

            steps += 1

        return -1
```
