# Problem Statement

Given an integer array `nums`, return the **Greatest Common Divisor (GCD)** of the **smallest** and **largest** numbers in the array.

The **Greatest Common Divisor (GCD)** of two numbers is the largest positive integer that divides both numbers without leaving a remainder.

---

## Example 1

```text
Input

nums = [2,5,6,9,10]
```

Output

```text
2
```

### Explanation

Smallest number

```text
2
```

Largest number

```text
10
```

Their GCD is

```text
GCD(2,10) = 2
```

## Example 3

```text
Input

nums = [3,3]
```

Output

```text
3
```

Both the smallest and largest numbers are

```text

```

Therefore

```text
GCD(3,3) = 3
```

---

## Constraints

```text
2 <= nums.length <= 1000

1 <= nums[i] <= 1000
```

---

# Python Solution

```python
class Solution:
    def findGCD(self, nums):
        mn = min(nums)
        mx = max(nums)

        while mx:
            mn, mx = mx, mn % mx

        return mn
```
