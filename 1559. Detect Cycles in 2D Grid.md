# Problem Statement

You are given a 2D grid of characters of size `m x n`.

Your task is to determine whether there exists a **cycle** formed by cells having the **same character**.

---

## 🔁 Cycle Definition

A cycle is a path that:
- Has length **≥ 4**
- Starts and ends at the **same cell**
- Moves only in **4 directions**:
  - Up
  - Down
  - Left
  - Right
- Moves only through cells having the **same character**
- Does **not revisit the immediate previous cell**

---

## 🎯 Objective

Return:

- `true` → if there exists any cycle  
- `false` → otherwise  

---

## 🔍 Examples

### Example 1


Input:
grid = [
["a","a","a","a"],
["a","b","b","a"],
["a","b","b","a"],
["a","a","a","a"]
]

Output:
true


---

### Example 2


Input:
grid = [
["c","c","c","a"],
["c","d","c","c"],
["c","c","e","c"],
["f","c","c","c"]
]

Output:
true


---

### Example 3


Input:
grid = [
["a","b","b"],
["b","z","b"],
["b","b","a"]
]

Output:
false


---

## ⚙️ Constraints

- `m == grid.length`
- `n == grid[i].length`
- `1 <= m, n <= 500`
- Grid contains only **lowercase English letters**
