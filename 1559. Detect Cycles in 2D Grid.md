# Problem Statement

You are given a 2D grid of characters of size `m x n`.

Your task is to determine whether there exists a **cycle** formed by cells having the **same character**.

---

## 🔁 Cycle Definition

A cycle is a path that:
- Has length **≥ 4**
- Starts and ends at the **same cell**
- Moves only in **4 directions**:
  - Up
  - Down
  - Left
  - Right
- Moves only through cells having the **same character**
- Does **not revisit the immediate previous cell**

---

## 🎯 Objective

Return:

- `true` → if there exists any cycle  
- `false` → otherwise  

---

## 🔍 Examples

### Example 1


Input:
grid = [
["a","a","a","a"],
["a","b","b","a"],
["a","b","b","a"],
["a","a","a","a"]
]

Output:
true


---

### Example 2


Input:
grid = [
["c","c","c","a"],
["c","d","c","c"],
["c","c","e","c"],
["f","c","c","c"]
]

Output:
true


---

### Example 3


Input:
grid = [
["a","b","b"],
["b","z","b"],
["b","b","a"]
]

Output:
false


---

## ⚙️ Constraints

- `m == grid.length`
- `n == grid[i].length`
- `1 <= m, n <= 500`
- Grid contains only **lowercase English letters**

---

## 💡 Solution

### 🔹 Approach: DFS (Cycle Detection in Grid)

#### 🧠 Key Idea

- Traverse each cell using **DFS**
- Track:
  - Visited cells
  - Previous cell (to avoid going back immediately)
- If we revisit a visited cell **not equal to parent**, a cycle exists

---

### 💻 Code

```python
class Solution:
    def containsCycle(self, grid):
        m, n = len(grid), len(grid[0])
        visited = [[False] * n for _ in range(m)]

        def dfs(x, y, px, py):
            if visited[x][y]:
                return True

            visited[x][y] = True
            directions = [(0,1), (1,0), (0,-1), (-1,0)]

            for dx, dy in directions:
                nx, ny = x + dx, y + dy

                if 0 <= nx < m and 0 <= ny < n and grid[nx][ny] == grid[x][y]:
                    if nx == px and ny == py:
                        continue
                    if dfs(nx, ny, x, y):
                        return True

            return False

        for i in range(m):
            for j in range(n):
                if not visited[i][j]:
                    if dfs(i, j, -1, -1):
                        return True

        return False
