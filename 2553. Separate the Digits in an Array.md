# Problem Statement

Given an array of positive integers `nums`, return an array `answer` that consists of the digits of each integer in `nums` after separating them in the same order they appear.

Separating the digits of an integer means extracting all of its digits in order.

Example:
```text
10921 → [1,0,9,2,1]
```

---

## 🧪 Examples

### Example 1
```text
Input:
nums = [13,25,83,77]

Output:
[1,3,2,5,8,3,7,7]
```

Explanation:
```text
13 → [1,3]
25 → [2,5]
83 → [8,3]
77 → [7,7]
```

---

### Example 2
```text
Input:
nums = [7,1,3,9]

Output:
[7,1,3,9]
```

Explanation:
```text
Each number already contains a single digit.
```

---

## 📌 Constraints

```text
1 <= nums.length <= 1000
1 <= nums[i] <= 10^5
```

---

## 🧠 Approach

### 💡 Key Idea

For every number in `nums`:
1. Convert number to string
2. Traverse each character
3. Convert back to integer
4. Add to answer array

This preserves the original digit order.

---

## 💻 Solution (Python)

```python
class Solution:
    def separateDigits(self, nums):
        ans = []

        for num in nums:
            for digit in str(num):
                ans.append(int(digit))

        return ans
```
