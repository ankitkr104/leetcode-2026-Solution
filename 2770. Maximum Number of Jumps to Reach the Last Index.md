# Problem Statement

You are given a `0-indexed` integer array `nums` and an integer `target`.

You are initially positioned at index `0`.

In one step, you can jump from index `i` to any index `j` such that:

```text
0 <= i < j < n
-target <= nums[j] - nums[i] <= target
```

Return the **maximum number of jumps** you can make to reach index `n - 1`.

If there is no way to reach index `n - 1`, return `-1`.

---

## 🧪 Examples

### Example 1
```text
Input:
nums = [1,3,6,4,1,2]
target = 2

Output:
3
```

Explanation:
```text
0 → 1 → 3 → 5
```

---

### Example 2
```text
Input:
nums = [1,3,6,4,1,2]
target = 3

Output:
5
```

Explanation:
```text
0 → 1 → 2 → 3 → 4 → 5
```

---

### Example 3
```text
Input:
nums = [1,3,6,4,1,2]
target = 0

Output:
-1
```

Explanation:
```text
No valid path exists.
```

---

## 📌 Constraints

```text
2 <= nums.length == n <= 1000
-10^9 <= nums[i] <= 10^9
0 <= target <= 2 * 10^9
```

---

## 🧠 Approach

Use **Dynamic Programming**.

Define:

```text
dp[i] = maximum jumps needed to reach index i
```

Initialize:
```text
dp[0] = 0
all others = -1
```

For every pair `(j, i)` where `j < i`:

- Check if jump is valid:
```text
abs(nums[i] - nums[j]) <= target
```

- Update:
```text
dp[i] = max(dp[i], dp[j] + 1)
```

---

## 💻 Solution (Python)

```python
class Solution:
    def maximumJumps(self, nums, target):
        n = len(nums)

        dp = [-1] * n
        dp[0] = 0

        for i in range(1, n):
            for j in range(i):
                if dp[j] != -1 and abs(nums[i] - nums[j]) <= target:
                    dp[i] = max(dp[i], dp[j] + 1)

        return dp[-1]
```
