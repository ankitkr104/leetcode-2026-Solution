# Problem Statement

You are given a binary string `s` of length `n`, where:

- `'1'` represents an **active** section.
- `'0'` represents an **inactive** section.

You may perform **at most one trade** on any queried substring.

A trade consists of two operations:

1. Convert a contiguous block of `'1'`s that is **surrounded by `'0'`s** into all `'0'`s.
2. Then convert a contiguous block of `'0'`s that is **surrounded by `'1'`s** into all `'1'`s.

You are also given an array `queries`, where

```text
queries[i] = [l, r]
```

represents the substring

```text
s[l...r]
```

For every query:

- Consider only the substring `s[l...r]`.
- Augment it with `'1'` on both sides:

```text
t = "1" + s[l...r] + "1"
```

- The added `'1'`s are only used for determining surrounded blocks and are **not counted** in the final answer.

Return an array where each element is the maximum possible number of active sections after performing at most one trade for the corresponding query.

---

## Example 1

```text
Input

s = "01"

queries = [[0,1]]
```

Output

```text
[1]
```

### Explanation

There is no `'1'` block surrounded by `'0'`s.

No valid trade exists.

---

## Example 2

```text
Input

s = "0100"

queries = [[0,3],[0,2],[1,3],[2,3]]
```

Output

```text
[4,3,1,1]
```

### Explanation

For query `[0,3]`

```text
Substring = "0100"

↓

Trade

↓

1111
```

Maximum active sections

```text
4
```

---

## Example 3

```text
Input

s = "1000100"

queries = [[1,5],[0,6],[0,4]]
```

Output

```text
[6,7,2]
```

---

## Example 4

```text
Input

s = "01010"

queries = [[0,3],[1,4],[1,3]]
```

Output

```text
[4,4,2]
```

---

## Constraints

```text
1 <= n <= 100000

1 <= queries.length <= 100000

s[i] ∈ {'0','1'}

0 <= l <= r < n
```

# Python Solution

```python
from bisect import bisect_left, bisect_right
from typing import List


class SegmentTree:
    def __init__(self, arr):
        self.n = len(arr)
        self.arr = arr
        self.seg = [0] * (self.n << 2)

        if self.n:
            self.build(1, 0, self.n - 1)

    def build(self, p, l, r):
        if l == r:
            self.seg[p] = self.arr[l]
            return

        mid = (l + r) >> 1

        self.build(p << 1, l, mid)
        self.build(p << 1 | 1, mid + 1, r)

        self.seg[p] = max(self.seg[p << 1], self.seg[p << 1 | 1])

    def query(self, L, R):
        if L > R:
            return 0

        def dfs(p, l, r):
            if L <= l and r <= R:
                return self.seg[p]

            mid = (l + r) >> 1
            ans = 0

            if L <= mid:
                ans = max(ans, dfs(p << 1, l, mid))

            if R > mid:
                ans = max(ans, dfs(p << 1 | 1, mid + 1, r))

            return ans

        return dfs(1, 0, self.n - 1)


class Solution:
    def maxActiveSectionsAfterTrade(self, s: str, queries: List[List[int]]) -> List[int]:

        n = len(s)

        prefix = [0] * (n + 1)

        for i in range(n):
            prefix[i + 1] = prefix[i] + (s[i] == '1')

        zeroBlocks = []
        blockLeft = []
        blockRight = []

        i = 0

        while i < n:

            st = i

            while i < n and s[i] == s[st]:
                i += 1

            if s[st] == '0':
                zeroBlocks.append(i - st)
                blockLeft.append(st)
                blockRight.append(i - 1)

        m = len(zeroBlocks)

        if m < 2:
            return [prefix[r + 1] - prefix[l] for l, r in queries]

        arr = [zeroBlocks[i] + zeroBlocks[i + 1] for i in range(m - 1)]

        seg = SegmentTree(arr)

        ans = []

        for l, r in queries:

            ones = prefix[r + 1] - prefix[l]

            i = bisect_left(blockRight, l)
            j = bisect_right(blockLeft, r) - 1

            if i > m - 1 or j < 0 or i >= j:
                ans.append(ones)
                continue

            firstLen = blockRight[i] - max(blockLeft[i], l) + 1
            lastLen = min(blockRight[j], r) - blockLeft[j] + 1

            if i + 1 == j:
                ans.append(ones + firstLen + lastLen)
                continue

            val1 = firstLen + zeroBlocks[i + 1]
            val2 = zeroBlocks[j - 1] + lastLen
            val3 = seg.query(i + 1, j - 2)

            ans.append(ones + max(val1, val2, val3))

        return ans
```
