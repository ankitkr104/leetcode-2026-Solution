## 📘 Problem Statement

You are given the head of a linked list.

Delete the **middle node** and return the head of the modified linked list.

The middle node of a linked list of size `n` is:

```text
⌊n / 2⌋
```

using **0-based indexing**.

### Examples

```text
n = 1 → middle index = 0
n = 2 → middle index = 1
n = 3 → middle index = 1
n = 4 → middle index = 2
n = 5 → middle index = 2
```

---

## 🧪 Examples

### Example 1

```text
Input:
head = [1,3,4,7,1,2,6]

Output:
[1,3,4,1,2,6]
```

### Explanation

```text
Indices:
0 1 2 3 4 5 6

Middle index = 3
Value = 7

Delete 7
```

Result:

```text
[1,3,4,1,2,6]
```

---

### Example 2

```text
Input:
head = [1,2,3,4]

Output:
[1,2,4]
```

### Explanation

```text
Middle index = 2
Value = 3

Delete 3
```

---

### Example 3

```text
Input:
head = [2,1]

Output:
[2]
```

### Explanation

```text
Middle index = 1
Value = 1

Delete 1
```

---

## 📌 Constraints

```text
1 <= Number of Nodes <= 10^5

1 <= Node.val <= 10^5
```
