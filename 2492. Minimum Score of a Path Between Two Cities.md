# Problem Statement

There are `n` cities numbered from `1` to `n`.

You are given an array

```text
roads[i] = [a, b, distance]
```

representing a **bidirectional** road between cities `a` and `b`.

The graph may not be fully connected.

The **score** of a path is defined as the **minimum edge weight** on that path.

Return the **minimum possible score** of any path from city

```text
1
```

to city

```text
n
```

A path may revisit cities and roads multiple times.

---

## Example 1

```text
Input:

n = 4

roads =
[[1,2,9],
 [2,3,6],
 [2,4,5],
 [1,4,7]]

Output:

5
```

Explanation

One optimal path is

```text
1 → 2 → 4
```

Road weights

```text
9

5
```

Score

```text
min(9,5)=5
```

---

## Example 2

```text
Input:

n = 4

roads =
[[1,2,2],
 [1,3,4],
 [3,4,7]]

Output:

2
```

Explanation

Since revisiting is allowed,

```text
1→2→1→3→4
```

Score

```text
min(2,2,4,7)=2
```

---

## Constraints

```text
2 <= n <= 100000

1 <= roads.length <= 100000
```

---

# Key Observation

The problem **does not ask for the shortest path.**

Instead,

```text
Score(path)

=

minimum edge on that path.
```

Also,

```text
You may revisit roads.
```

This changes everything.

---

# Python Solution

```python
from collections import defaultdict, deque

class Solution:
    def minScore(self, n, roads):

        graph = defaultdict(list)

        for u, v, d in roads:

            graph[u].append((v, d))
            graph[v].append((u, d))

        ans = float("inf")

        visited = set()

        q = deque([1])

        visited.add(1)

        while q:

            node = q.popleft()

            for nxt, dist in graph[node]:

                ans = min(ans, dist)

                if nxt not in visited:

                    visited.add(nxt)

                    q.append(nxt)

        return ans
```
