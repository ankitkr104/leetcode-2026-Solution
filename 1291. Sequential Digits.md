# Problem Statement

An integer has **sequential digits** if every digit is exactly one greater than the previous digit.

Examples:

```text
1234 ✓

4567 ✓

6789 ✓

1357 ✗

1245 ✗
```

Given two integers

```text
low

high
```

return a **sorted list** of all sequential digit numbers in the inclusive range

```text
[low, high]
```

---

## Example 1

```text
Input

low = 100

high = 300
```

Output

```text
[123,234]
```

### Explanation

Sequential numbers between

```text
100

300
```

are

```text
123

234
```

---

## Example 2

```text
Input

low = 1000

high = 13000
```

Output

```text
[1234,2345,3456,4567,5678,6789,12345]
```

---

## Constraints

```text
10 <= low <= high <= 10^9
```

---

# Python Solution

```python
class Solution:
    def sequentialDigits(self, low, high):

        digits = "123456789"

        ans = []

        for length in range(2, 10):

            for start in range(10 - length):

                num = int(digits[start:start + length])

                if low <= num <= high:

                    ans.append(num)

        return sorted(ans)
```
