# Problem Statement

You are given a string `s` consisting of lowercase English letters and the special characters:

```text
*
#
%
```

You are also given an integer:

```text
k
```

Build a string `result` by processing `s` from left to right according to the following rules:

### Operations

#### Lowercase Letter

```text
Append the letter to result.
```

#### '*'

```text
Remove the last character from result, if it exists.
```

#### '#'

```text
Duplicate the current result and append it to itself.

result = result + result
```

#### '%'

```text
Reverse the current result.
```

Return the character at index `k` of the final string.

If `k` is out of bounds, return:

```text
'.'
```

---

## 🧪 Example 1

```text
Input:
s = "a#b%*"
k = 1

Output:
"a"
```

### Explanation

```text
"a"
"aa"
"aab"
"baa"
"ba"
```

Final string:

```text
"ba"
```

Character at index 1:

```text
'a'
```

---

## 🧪 Example 2

```text
Input:
s = "cd%#*#"
k = 3

Output:
"d"
```

### Explanation

```text
"c"
"cd"
"dc"
"dcdc"
"dcd"
"dcddcd"
```

Character at index 3:

```text
'd'
```

---

## 🧪 Example 3

```text
Input:
s = "z*#"
k = 0

Output:
"."
```

### Explanation

```text
"z"
""
""
```

Final string is empty.

Index 0 is invalid.

---

## 📌 Constraints

```text
1 <= s.length <= 10^5

0 <= k <= 10^15

s consists only of:
'a' - 'z'
*
#
%

Length of final result <= 10^15
```

---
## 💻 Solution (Python)

```python
class Solution:
    def processStr(self, s, k):

        n = len(s)

        lengths = [0] * (n + 1)

        for i, ch in enumerate(s):

            cur = lengths[i]

            if 'a' <= ch <= 'z':
                lengths[i + 1] = cur + 1

            elif ch == '*':
                lengths[i + 1] = max(0, cur - 1)

            elif ch == '#':
                lengths[i + 1] = min(
                    10**15 + 1,
                    cur * 2
                )

            else:  # '%'
                lengths[i + 1] = cur

        if k >= lengths[n]:
            return '.'

        for i in range(n - 1, -1, -1):

            ch = s[i]

            prev_len = lengths[i]
            curr_len = lengths[i + 1]

            if 'a' <= ch <= 'z':

                if k == curr_len - 1:
                    return ch

            elif ch == '*':

                pass

            elif ch == '#':

                if prev_len > 0 and k >= prev_len:
                    k -= prev_len

            else:  # '%'

                if curr_len > 0:
                    k = curr_len - 1 - k

        return '.'
```
