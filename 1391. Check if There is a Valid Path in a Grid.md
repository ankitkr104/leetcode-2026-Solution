# Problem Statement

You are given an `m x n` grid where each cell represents a **street type**.

Each street connects specific directions:

| Type | Connections |
|------|------------|
| 1 | Left ↔ Right |
| 2 | Up ↔ Down |
| 3 | Left ↔ Down |
| 4 | Right ↔ Down |
| 5 | Left ↔ Up |
| 6 | Right ↔ Up |

---

## 🎯 Objective

You start at the **top-left cell (0, 0)** and want to reach the **bottom-right cell (m - 1, n - 1)**.

Return:
- `true` → if there exists a valid path  
- `false` → otherwise  

---

## 📌 Important Rule

You can move from one cell to another **only if both cells are connected**.

👉 That means:
- Current cell allows movement in that direction
- Next cell must allow movement **back to the current cell**

---

## 🔍 Examples

### Example 1


Input:
grid = [[2,4,3],[6,5,2]]

Output:
true


---

### Example 2


Input:
grid = [[1,2,1],[1,2,1]]

Output:
false


---

### Example 3


Input:
grid = [[1,1,2]]

Output:
false


---

## ⚙️ Constraints

- `1 <= m, n <= 300`
- `1 <= grid[i][j] <= 6`

---

#### 🧠 Key Idea

- Treat grid as a graph
- Move only if:
  - Direction is allowed from current cell
  - Reverse direction is allowed from next cell

---

directions = {

    4: [(0, 1), (1, 0)],    # right, down

    6: [(0, 1), (-1, 0)]    # right, up
