# Problem Statement

A shop offers the following discount:

```text
For every 2 candies purchased,
you can get 1 candy for free.
```

The free candy must satisfy:

```text
free_candy_cost <= min(cost of the two purchased candies)
```

Given an array:

```text
cost[i]
```

representing the cost of each candy, return the minimum cost required to buy all candies.

---

## 🧪 Examples

### Example 1

```text
Input:
cost = [1,2,3]

Output:
5
```

Explanation:

```text
Buy: 3 and 2
Free: 1

Total = 5
```

---

### Example 2

```text
Input:
cost = [6,5,7,9,2,2]

Output:
23
```

Explanation:

```text
Buy: 9,7
Free: 6

Buy: 5,2
Free: 2

Total = 23
```

---

### Example 3

```text
Input:
cost = [5,5]

Output:
10
```

---

## 📌 Constraints

```text
1 <= cost.length <= 100
1 <= cost[i] <= 100
```

---

## 💻 Solution (Python)

```python
class Solution(object):
    def minimumCost(self, cost):
        # Sort the candies from most expensive to cheapest
        cost.sort(reverse=True)
        
        total_cost = 0
        
        # Iterate through the candies, adding the pairs we must buy
        for i in range(len(cost)):
            # Skip every 3rd candy (index 2, 5, 8, ...)
            if (i + 1) % 3 != 0:
                total_cost += cost[i]
                
        return total_cost
```
