# Problem Statement

A shop offers the following discount:

```text
For every 2 candies purchased,
you can get 1 candy for free.
```

The free candy must satisfy:

```text
free_candy_cost <= min(cost of the two purchased candies)
```

Given an array:

```text
cost[i]
```

representing the cost of each candy, return the minimum cost required to buy all candies.

---

## 🧪 Examples

### Example 1

```text
Input:
cost = [1,2,3]

Output:
5
```

Explanation:

```text
Buy: 3 and 2
Free: 1

Total = 5
```

---

### Example 2

```text
Input:
cost = [6,5,7,9,2,2]

Output:
23
```

Explanation:

```text
Buy: 9,7
Free: 6

Buy: 5,2
Free: 2

Total = 23
```

---

### Example 3

```text
Input:
cost = [5,5]

Output:
10
```

---

## 📌 Constraints

```text
1 <= cost.length <= 100
1 <= cost[i] <= 100
```
