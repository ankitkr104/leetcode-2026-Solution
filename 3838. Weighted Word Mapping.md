# Problem Statement

You are given:

- An array of strings `words`
- An integer array `weights` of length `26`, where:

```text
weights[i]
```

represents the weight of the:

```text
(i-th) lowercase letter
```

The weight of a word is the sum of the weights of all its characters.

For each word:

1. Compute its total weight.
2. Take:

```text
weight % 26
```

3. Convert the result to a lowercase letter using **reverse alphabetical order**:

```text
0  -> z
1  -> y
2  -> x
...
25 -> a
```

Return the concatenation of all mapped characters.

---

## 🧪 Example 1

```text
Input:

words = ["abcd","def","xyz"]

weights =
[5,3,12,14,1,2,3,2,10,6,6,9,7,8,7,10,8,9,6,9,9,8,3,7,7,2]

Output:

"rij"
```

### Explanation

```text
"abcd"
= 5 + 3 + 12 + 14
= 34

34 % 26 = 8

8 -> 'r'
```

```text
"def"
= 14 + 1 + 2
= 17

17 % 26 = 17

17 -> 'i'
```

```text
"xyz"
= 7 + 7 + 2
= 16

16 % 26 = 16

16 -> 'j'
```

Result:

```text
"rij"
```

## 📌 Constraints

```text
1 <= words.length <= 100

1 <= words[i].length <= 10

weights.length == 26

1 <= weights[i] <= 100

words[i] consists only of lowercase English letters
```

---

## 💻 Solution (Python)

```python
class Solution:
    def mapWordWeights(self, words, weights):

        ans = []

        for word in words:

            total = 0

            for ch in word:
                total += weights[ord(ch) - ord('a')]

            mod = total % 26

            ans.append(chr(ord('z') - mod))

        return "".join(ans)
```
