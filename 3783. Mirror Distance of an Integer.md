## Problem Statement

You are given an integer `n`.

Define the **mirror distance** as:

```text id="v9k3x1"
|n - reverse(n)|
```

Where:

* `reverse(n)` is the integer formed by reversing the digits of `n`
* Leading zeros are ignored after reversing

---

## Goal

Return the **mirror distance** of the integer `n`.

---

## Examples

### Example 1

```text id="f2l9wp"
Input:
n = 25

Output:
27
```

**Explanation:**

```text id="n0d4xk"
reverse(25) = 52
|25 - 52| = 27
```

---

### Example 2

```text id="h7q1vd"
Input:
n = 10

Output:
9
```

**Explanation:**

```text id="g5z3rm"
reverse(10) = 01 → 1
|10 - 1| = 9
```

---

### Example 3

```text id="k2m8sj"
Input:
n = 7

Output:
0
```

**Explanation:**

```text id="w4c6ty"
reverse(7) = 7
|7 - 7| = 0
```

---

## Constraints

* `1 <= n <= 10^9`

---

---

## Solution

* `class Solution:
    def mirrorDistance(self, n):
        rev = int(str(n)[::-1])
        return abs(n - rev)`
