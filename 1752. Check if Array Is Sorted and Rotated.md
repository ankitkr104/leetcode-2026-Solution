# Problem Statement

Given an array `nums`, return `true` if the array was originally sorted in non-decreasing order and then rotated some number of times (possibly zero). Otherwise, return `false`.

Duplicates may exist.

A rotation means:
```text
A[i] == original[(i + x) % n]
```

for some rotation count `x`.

---

## 🧪 Examples

### Example 1
```text
Input:
nums = [3,4,5,1,2]

Output:
true
```

Explanation:
```text
Original sorted array:
[1,2,3,4,5]

After rotation:
[3,4,5,1,2]
```

---

### Example 2
```text
Input:
nums = [2,1,3,4]

Output:
false
```

---

### Example 3
```text
Input:
nums = [1,2,3]

Output:
true
```

Explanation:
```text
Already sorted (0 rotations)
```

---

## 📌 Constraints

```text
1 <= nums.length <= 100
1 <= nums[i] <= 100
```

---

## 🧠 Approach

### 💡 Key Insight

In a sorted and rotated array:
```text
there can be at most ONE drop
```

Meaning:
```text
nums[i] > nums[i+1]
```

can happen only once.

Also check:
```text
last element > first element
```

because rotation is circular.

---

## 💻 Solution (Python)

```python
class Solution:
    def check(self, nums):

        count = 0
        n = len(nums)

        for i in range(n):

            if nums[i] > nums[(i + 1) % n]:
                count += 1

        return count <= 1
```
