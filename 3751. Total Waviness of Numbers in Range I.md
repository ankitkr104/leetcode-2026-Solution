## 📘 Problem Statement

You are given two integers:

```text
num1, num2
```

representing the inclusive range:

```text
[num1, num2]
```

The **waviness** of a number is the count of its:

- Peaks
- Valleys

### Peak

A digit is a peak if:

```text
digit > left neighbor
AND
digit > right neighbor
```

### Valley

A digit is a valley if:

```text
digit < left neighbor
AND
digit < right neighbor
```

Notes:

```text
First and last digits cannot be peaks or valleys.
Numbers with fewer than 3 digits have waviness = 0.
```

Return the total waviness of all numbers in the range.

---

## 🧪 Examples

### Example 1

```text
Input:
num1 = 120
num2 = 130

Output:
3
```

Explanation:

```text
120 → 1
121 → 1
130 → 1

Total = 3
```

---

### Example 2

```text
Input:
num1 = 198
num2 = 202

Output:
3
```

Explanation:

```text
198 → peak at 9
201 → valley at 0
202 → valley at 0

Total = 3
```

---

### Example 3

```text
Input:
num1 = 4848
num2 = 4848

Output:
2
```

Explanation:

```text
4 8 4 8

8 = peak
4 = valley

waviness = 2
```

---

## 📌 Constraints

```text
1 <= num1 <= num2 <= 10^5
```

---

## 🧠 Approach

### 💡 Observation

Maximum number:

```text
10^5
```

Range size is manageable.

For every number:

1. Convert to string
2. Check every middle digit
3. Count peaks and valleys
4. Add to answer
