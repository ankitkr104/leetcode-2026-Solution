# Problem Statement

An integer `x` is considered **good** if after rotating each of its digits by **180 degrees**, we get a **valid number** that is **different from x**.

### 🔁 Digit Rotation Rules:
- `0 → 0`, `1 → 1`, `8 → 8` (remain same)
- `2 ↔ 5`, `6 ↔ 9` (change to different digits)
- `3, 4, 7` → ❌ invalid (cannot be rotated)

A number is:
- ✅ **Valid** if all digits can be rotated
- ⭐ **Good** if it is valid AND changes after rotation

---

## 🎯 Objective

Given an integer `n`, return the number of **good integers** in the range `[1, n]`.

---

## 🧠 Approach

We iterate through all numbers from `1` to `n` and:

1. Check each digit:
   - If it contains `3, 4, 7` → ❌ invalid number → skip
2. Track if at least one digit is from `{2, 5, 6, 9}`:
   - This ensures the number **changes after rotation**
3. Count such numbers

---

## ⚙️ Algorithm

- Loop from `1` to `n`
- For each number:
  - Convert to string
  - Check digits:
    - If invalid digit → break
    - If changing digit → mark as "good candidate"
- If valid AND changed → increment count

---
---

## 📌 Constraints

- `1 <= n <= 10^4`

---
## 🧾 Solution (Python)

```python
class Solution:
    def rotatedDigits(self, n):
        count = 0
        
        for num in range(1, n + 1):
            s = str(num)
            is_valid = True
            is_good = False
            
            for ch in s:
                if ch in '347':
                    is_valid = False
                    break
                if ch in '2569':
                    is_good = True
            
            if is_valid and is_good:
                count += 1
        
        return count
