# Problem Statement

You are given an integer `n`.

Construct a new integer `x` by concatenating all the **non-zero digits** of `n` in their original order.

- If `n` contains no non-zero digits, then

```text
x = 0
```

Let

```text
sum
```

be the sum of the digits of `x`.

Return

```text
x × sum
```

---

## Example 1

```text
Input:

n = 10203004
```

Output

```text
12340
```

### Explanation

Non-zero digits:

```text
1 2 3 4
```

Concatenated number:

```text
x = 1234
```

Digit sum:

```text
1 + 2 + 3 + 4 = 10
```

Answer:

```text
1234 × 10 = 12340
```

---

## Example 2

```text
Input:

n = 1000
```

Output

```text
1
```

### Explanation

Non-zero digits:

```text
1
```

Therefore

```text
x = 1
```

Digit sum:

```text
1
```

Answer:

```text
1 × 1 = 1
```

---

## Constraints

```text
0 <= n <= 10^9
```

---

# Python Solution

```python
class Solution:
    def concatSum(self, n):

        x = 0
        digitSum = 0

        for ch in str(n):

            d = int(ch)

            if d != 0:

                x = x * 10 + d

                digitSum += d

        return x * digitSum
```
