## 🧩 Problem Statement

You are given a **circular array** `nums` and an array `queries`.

For each query `i`, you need to find:

* The **minimum distance** between the element at index `queries[i]` and **any other index `j`** such that:

```text
nums[j] == nums[queries[i]]
```

* If no such index exists, return:

```text
-1
```

---

## 🔄 Circular Array Definition

The array is **circular**, meaning:

* Next index:

  ```text
  (i + 1) % n
  ```
* Previous index:

  ```text
  (i - 1 + n) % n
  ```

---

## 🎯 Goal

Return an array `answer` such that:

```text
answer[i] = minimum distance for query i
```

---

## 📌 Examples

### Example 1

```text
Input:
nums = [1,3,1,4,1,3,2]
queries = [0,3,5]

Output:
[2,-1,3]
```

**Explanation:**

* Query 0:

  * nums[0] = 1
  * nearest same value at index 2 → distance = 2

* Query 1:

  * nums[3] = 4
  * no other occurrence → -1

* Query 2:

  * nums[5] = 3
  * nearest index = 1
  * circular path → distance = 3

---

### Example 2

```text
Input:
nums = [1,2,3,4]
queries = [0,1,2,3]

Output:
[-1,-1,-1,-1]
```

**Explanation:**

* All elements are unique → no valid pairs

---

## ⚙️ Constraints

* `1 <= queries.length <= nums.length <= 10^5`
* `1 <= nums[i] <= 10^6`
* `0 <= queries[i] < nums.length`

---


## Solution (Python)

```python
import bisect
from collections import defaultdict

class Solution:
    def solveQueries(self, nums, queries):
        n = len(nums)
        
        pos = defaultdict(list)
        for i, x in enumerate(nums):
            pos[x].append(i)
        
        res = []
        
        for q in queries:
            arr = pos[nums[q]]
            
            if len(arr) == 1:
                res.append(-1)
                continue
            
            idx = bisect.bisect_left(arr, q)
            
            ans = float('inf')
            
            # previous (circular)
            prev_idx = arr[idx - 1] if idx > 0 else arr[-1]
            
            # next (circular)
            next_idx = arr[idx + 1] if idx < len(arr) - 1 else arr[0]
            
            # compute distances
            for j in [prev_idx, next_idx]:
                d = abs(q - j)
                ans = min(ans, min(d, n - d))
            
            res.append(ans)
        
        return res
```
