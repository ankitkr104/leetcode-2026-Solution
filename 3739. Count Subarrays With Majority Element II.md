## Problem Statement

You are given an integer array `nums` and an integer `target`.

Return the number of subarrays in which `target` is the **majority element**.

A majority element is defined as an element that appears **strictly more than half** of the total elements in the subarray.

Return the total number of such subarrays.

---

## Example 1

```text
Input:
nums = [1,2,2,3]
target = 2

Output:
5
```

### Explanation

The valid subarrays are:

```text
[2]
[2]
[2,2]
[1,2,2]
[2,2,3]
```

Hence the answer is:

```text
5
```

---

## Example 2

```text
Input:
nums = [1,1,1,1]
target = 1

Output:
10
```

### Explanation

Every subarray has `1` as the majority element.

---

## Example 3

```text
Input:
nums = [1,2,3]
target = 4

Output:
0
```

### Explanation

The target never appears, so no valid subarray exists.

---

## Constraints

```text
1 <= nums.length <= 10^5

1 <= nums[i] <= 10^9

1 <= target <= 10^9
```

---

# Python Solution

```python
from bisect import bisect_left

class Fenwick:
    def __init__(self, n):
        self.bit = [0] * (n + 1)

    def update(self, i, val):
        while i < len(self.bit):
            self.bit[i] += val
            i += i & -i

    def query(self, i):
        s = 0
        while i > 0:
            s += self.bit[i]
            i -= i & -i
        return s


class Solution:
    def countMajoritySubarrays(self, nums, target):

        pref = [0]

        s = 0
        for x in nums:
            if x == target:
                s += 1
            else:
                s -= 1
            pref.append(s)

        vals = sorted(set(pref))

        bit = Fenwick(len(vals))

        ans = 0

        for x in pref:

            idx = bisect_left(vals, x) + 1

            ans += bit.query(idx - 1)

            bit.update(idx, 1)

        return ans
```
