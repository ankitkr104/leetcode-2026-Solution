# Problem Statement

You are given an integer array `nums`.

Replace every element with the **sum of its digits**.

Return the **minimum element** in the array after all replacements.

---

## 🧪 Examples

### Example 1
```text
Input:
nums = [10,12,13,14]

Output:
1
```

Explanation:
```text
10 → 1
12 → 3
13 → 4
14 → 5

New array:
[1,3,4,5]

Minimum = 1
```

---

### Example 2
```text
Input:
nums = [1,2,3,4]

Output:
1
```

Explanation:
```text
Digit sums:
[1,2,3,4]

Minimum = 1
```

---

### Example 3
```text
Input:
nums = [999,19,199]

Output:
10
```

Explanation:
```text
999 → 27
19 → 10
199 → 19

New array:
[27,10,19]

Minimum = 10
```

---

## 📌 Constraints

```text
1 <= nums.length <= 100
1 <= nums[i] <= 10^4
```

---

## 🧠 Approach

### 💡 Key Insight

For each number:
1. Calculate the sum of its digits
2. Keep track of the minimum digit sum

---

## 💻 Solution (Python)

```python
class Solution:
    def minElement(self, nums):

        ans = float('inf')

        for num in nums:

            digit_sum = 0

            while num:
                digit_sum += num % 10
                num //= 10

            ans = min(ans, digit_sum)

        return ans
```
