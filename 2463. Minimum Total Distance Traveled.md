# 2463. Minimum Total Distance Traveled

## Problem Statement

There are some robots and factories on the X-axis.

* You are given an array `robot` where `robot[i]` is the position of the ith robot.
* You are also given a 2D array `factory` where:

  * `factory[j][0]` = position of factory
  * `factory[j][1]` = maximum number of robots it can repair

### Rules

* Each robot can move left or right.
* When a robot reaches a factory (with available capacity), it gets repaired and stops.
* Robots can cross each other.
* Factories with full capacity are ignored.

### Goal

Return the **minimum total distance** traveled by all robots.

---

## Example

### Example 1:

```
Input:
robot = [0,4,6]
factory = [[2,2],[6,2]]

Output:
4
```

### Example 2:

```
Input:
robot = [1,-1]
factory = [[-2,1],[2,1]]

Output:
2
```

---

## Key Insights

* Direction doesn't matter → only **distance matters**
* Sort both robots and factories
* Assign robots to factories in order (no crossing)
* Use **Dynamic Programming (DP)**

---

## Approach

### Step 1: Sort

```
robot.sort()
factory.sort()
```

---

### Step 2: DP Definition

```
dp[i][j] = minimum cost to fix first i robots using first j factories
```

---

### Step 3: Transition

For each factory, assign `k` robots:

```
dp[i][j] = min(
    dp[i][j-1],  # skip factory
    dp[i-k][j-1] + cost
)
```

---

## Solution Code

```python
class Solution:
    def minimumTotalDistance(self, robot, factory):
        robot.sort()
        factory.sort()
        
        n = len(robot)
        m = len(factory)
        
        INF = float('inf')
        dp = [[INF]*(m+1) for _ in range(n+1)]
        dp[0][0] = 0
        
        for j in range(1, m+1):
            pos, limit = factory[j-1]
            
            for i in range(n+1):
                # Option 1: skip this factory
                dp[i][j] = dp[i][j-1]
                
                # Option 2: assign k robots
                cost = 0
                for k in range(1, min(limit, i) + 1):
                    cost += abs(robot[i-k] - pos)
                    dp[i][j] = min(dp[i][j], dp[i-k][j-1] + cost)
        
        return dp[n][m]
```

---

## Complexity

```
Time Complexity:  O(n * m * limit)
Space Complexity: O(n * m)
```

---

## Intuition

* Treat factories as **groups with capacity**
* Assign closest robots first
* Use DP to try all valid assignments efficiently

---

## Pattern

This problem follows:

```
Sorting + DP + Capacity Assignment
```
